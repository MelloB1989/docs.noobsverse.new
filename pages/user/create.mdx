## Overview

The `create` API endpoint allows you to create a new user subscription.

## HTTP Request

### POST

```
POST http://localhost:3000/v1/user_subscription/create
```

- **Authorization**: Bearer `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiIwXzZpaGFjanJyIn0.Rf1ZUJpJdBBbep4cGCqwBQ2A8IyETp2UTrnecGHdi6I`

## Body Parameters

The body of the request should be sent as JSON with the following fields:

| Field            | Type   | Description                                 |
|------------------|--------|---------------------------------------------|
| subscription_id  | string | Unique identifier for the subscription.     |
| start_date       | string | Start date of the subscription (dd/mm/yyyy).|
| end_date         | string | End date of the subscription (dd/mm/yyyy).  |

### Example Request Body

```json
{
  "subscription_id": "ftx0li8efd",
  "start_date": "27/11/2004",
  "end_date": "27/11/2004"
}
```
