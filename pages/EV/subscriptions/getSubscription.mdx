## Overview

The `getSubscription` API is an HTTP-based service that allows users to retrieve subscription information. It is secured with Bearer authentication and is accessible at a specified endpoint URL.

## Endpoint

- **URL**: `http://localhost:3000/v1/subscriptions/get`
- **Method**: `GET`

Each subscription object contains the following fields:

- **id**: Unique identifier for the subscription.
- **monthly_fee**: Cost of the subscription per month.
- **discount**: Discount applied to the subscription.
- **name**: Name of the subscription plan.
- **bike_id**: Identifier for the associated bike.
- **type**: The type of subscription, indicating whether it is monthly or annual.
