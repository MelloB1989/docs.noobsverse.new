# Get Subscription API Documentation

## Overview

The `getSubscription` API allows you to retrieve subscription details. You need to include the required headers and authentication token to make a successful request.

## Endpoint

**URL:** `http://localhost:3000/v1/subscriptions/get`  
**Method:** `GET`  
**Request Body:** JSON  
**Authentication:** Bearer Token

## Notes

- Ensure that the `X-Karma-Admin-Auth` header and the Bearer token are valid and match the required format.
- The required fields for each subscription record are `id`, `monthly_fee`, `discount`, `name`, `bike_id`, and `type`.
- The `type` field specifies the subscription type and can be either `monthly` or `annual`.

For any further questions or support, please contact the API administration team.
