# getSubscription API

Retrieve a detailed list of available subscription plans.

## Response

The response will contain the subscription details in JSON format. Each subscription will include the following fields:

- `id`: Unique identifier for the subscription.
- `monthly_fee`: Monthly fee for the subscription.
- `discount`: Discount applied to the subscription.
- `name`: Name of the subscription plan.
- `bike_id`: Identifier for the associated bike.
- `type`: Subscription type, e.g., `monthly` or `annual`.
