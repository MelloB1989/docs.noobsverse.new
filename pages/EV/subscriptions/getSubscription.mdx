# Get Subscription API

This documentation provides details for the `getSubscription` API, an HTTP API that retrieves a list of subscription plans.

## Endpoint

- **URL:** `http://localhost:3000/v1/subscriptions/get`
- **Method:** GET
- **Content-Type:** `application/json`
- **Authorization:** Bearer Token

## Response

The API will respond with a JSON array of available subscription plans. Each object in the array will contain the following fields:

- `id`: Unique identifier for the subscription.
- `monthly_fee`: The monthly fee for the subscription plan.
- `discount`: The discount amount for the subscription plan.
- `name`: The name of the subscription plan.
- `bike_id`: Identifier for the associated bike.
- `type`: Type of subscription, e.g., monthly or annual.

Ensure that you include the correct bearer token and headers in your API requests to successfully interact with the `getSubscription` API.
