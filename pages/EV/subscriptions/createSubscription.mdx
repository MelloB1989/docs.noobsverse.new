# createSubscription

The `createSubscription` API endpoint allows for the creation of new subscription plans. Below are the details needed to make a request to this endpoint.

## Endpoint

**URL:** `http://localhost:3000/v1/subscriptions/create`  
**Type:** `POST`  
**Body format:** `JSON`  
**Auth:** None required

### Fields

- **id**: (String) A unique identifier for the subscription.
- **monthly_fee**: (Number) Monthly fee for the subscription.
- **discount**: (Number) Discount applied to the subscription.
- **name**: (String) Name of the subscription plan.
- **bike_id**: (String) Identifier for the bike associated with the subscription.
- **type**: (String) Type of subscription (e.g., "monthly" or "annual").
- **security_deposit**: (Number) Security deposit for the subscription.

Use this information to interact with the `createSubscription` endpoint, ensuring that the structure and details of your request adhere to the specified format.
