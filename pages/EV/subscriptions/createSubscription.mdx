# Create Subscription API

## Endpoint

**POST** `http://localhost:3000/v1/subscriptions/create`

## Authorization

No authorization required.


## Description

This API endpoint allows the creation of new subscription plans. Each subscription plan contains the following fields:

- **id**: Unique identifier for the subscription.
- **monthly_fee**: The monthly fee for the subscription.
- **discount**: Any discount applied.
- **name**: The name of the subscription plan.
- **bike_id**: The ID of the bike associated with the subscription.
- **type**: The type of subscription, either "monthly" or "annual".
- **security_deposit**: The security deposit amount required for the subscription.
