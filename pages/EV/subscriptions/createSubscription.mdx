# createSubscription API

## Endpoint Description

The `createSubscription` API endpoint is used to create new subscriptions. This endpoint expects a POST request with a JSON body containing the subscription details.

## Endpoint

- **URL:** `http://localhost:3000/v1/subscriptions/create`
- **Method:** `POST`

## Headers

The following headers should be included in the request:
```
X-Karma-Admin-Auth: CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab
```

### Fields Description

- **id**: (string) The unique identifier for the subscription.
- **monthly_fee**: (number) The monthly fee for the subscription.
- **discount**: (number) The discount applied to the monthly fee.
- **name**: (string) The name of the subscription plan.
- **bike_id**: (string) The identifier for the bike associated with the subscription.
- **type**: (string) The type of subscription. Could be `monthly` or `annual`.
- **security_deposit**: (number) The security deposit required for the subscription.

### Authentication

No authentication is required to access this endpoint.
