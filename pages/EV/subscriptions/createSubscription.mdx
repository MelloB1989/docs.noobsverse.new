## Create Subscription

### Endpoint
`POST http://localhost:3000/v1/subscriptions/create`

### Description
This endpoint allows the creation of new subscriptions.

### Header
```json
{
  "X-Karma-Admin-Auth": "CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab"
}
```

The request body should be in JSON format and contains a list of subscription objects.
