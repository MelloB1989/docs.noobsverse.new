# createSubscription API

The `createSubscription` API is used to create new subscription plans for bikes. It is an HTTP POST request and does not require authentication.

## Endpoint

`POST http://localhost:3000/v1/subscriptions/create`

## Headers

- `X-Karma-Admin-Auth`: `CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab`

## Request Body

The request body should be in JSON format and consist of an array of subscription objects. Each subscription object contains the following fields:

- `id` (string): Unique identifier for the subscription.
- `monthly_fee` (number): The monthly fee for the subscription plan.
- `discount` (number): The discount applied to the subscription plan.
- `name` (string): The name of the subscription plan.
- `bike_id` (string): The identifier for the bike associated with the subscription.
- `type` (string): The type of subscription, e.g., `monthly` or `annual`.
- `security_deposit` (number): The security deposit amount for the subscription.

## Usage

Use this API to create subscription plans for different bikes with varying fees, discounts, and types. No authentication is required to access this endpoint. Include the `X-Karma-Admin-Auth` header in your request.
