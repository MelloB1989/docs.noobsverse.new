# Create Subscription

## Overview

The `createSubscription` API endpoint is used to create new subscription plans.

## Endpoint

**URL:** `http://localhost:3000/v1/subscriptions/create`  
**Method:** `POST`  
**Body Format:** JSON  
**Authentication:** None

## Headers

```json
{
  "X-Karma-Admin-Auth": "CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab"
}
```

### Fields

- `id` (string): The unique identifier for the subscription plan.
- `monthly_fee` (number): The monthly fee for the subscription plan.
- `discount` (number): The discount applied to the monthly fee.
- `name` (string): The name of the subscription plan.
- `bike_id` (string): The unique identifier for the bike related to the subscription plan.
- `type` (string): The type of the subscription (e.g., `monthly`, `annual`).
- `security_deposit` (number): The security deposit required for the subscription plan.
