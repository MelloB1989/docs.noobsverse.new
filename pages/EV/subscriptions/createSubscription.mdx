# createSubscription API

## Overview

The `createSubscription` API is used to create new subscription plans. This endpoint does not require any authentication.

## Endpoint

**POST** `http://localhost:3000/v1/subscriptions/create`

## Headers

- `X-Karma-Admin-Auth`: `CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab`

## Request Body

The request body should be in JSON format containing an array of subscription objects. Each subscription should have the following fields:

- `id` (string): Unique identifier for the subscription plan.
- `monthly_fee` (number): The monthly fee for the subscription.
- `discount` (number): Discount amount on the monthly fee.
- `name` (string): Name of the subscription plan.
- `bike_id` (string): Identifier for the associated bike.
- `type` (string): Type of subscription (`"monthly"` or `"annual"`).
- `security_deposit` (number): Security deposit amount for the subscription.
