## verifyOTP

### HTTP Method

`POST`

### URL

`http://localhost:3000/v1/auth/verify_otp`

### Authentication

None

### Request Body

#### Content-Type

`application/json`

#### Example

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

### Description

The `verifyOTP` API is used to verify the OTP sent to a user's phone number. This API endpoint expects a `POST` request with the phone number and OTP in the JSON body. The server processes the request and verifies the provided OTP.

### Response

The structure and content of the response depend on the server's implementation and are not detailed in the provided information.
