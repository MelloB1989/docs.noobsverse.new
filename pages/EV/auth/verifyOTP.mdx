## verifyOTP API

### Overview

This API endpoint is used to verify the OTP (One-Time Password) sent to a user's phone number.

### Endpoint

- **URL**: `http://localhost:3000/v1/auth/verify_otp`
- **Method**: `POST`
- **Body Format**: `JSON`
- **Authentication**: None

### Request Body

The request body should be in JSON format and include the following parameters:

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

### Parameters

- `phone` (string): The phone number to which the OTP was sent. Example: `"+919812940706"`.
- `otp` (string): The OTP received by the user. Example: `"815897"`.

### Example Request

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

### Response

The API will return a response indicating the success or failure of the OTP verification. The details of the response are not specified in this documentation.
