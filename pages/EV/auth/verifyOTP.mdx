## Overview

The `verifyOTP` endpoint is used to verify the OTP (One Time Password) for a given phone number. This is part of the authentication process.

## Endpoint

- **Type**: HTTP
- **Sequence**: 2

## HTTP Request

### Method: POST

- **URL**: `http://localhost:3000/v1/auth/verify_otp`

### Authentication

- **Auth**: None

## Request Body

The request body should be in JSON format. The following parameters must be included:

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

| Parameter | Type   | Description                |
|-----------|--------|----------------------------|
| `phone`   | String | The phone number with country code to verify the OTP for. |
| `otp`     | String | The OTP (One Time Password) to be verified for the phone number. |

## Example Usage

Here's an example on how to use the `verifyOTP` endpoint:
