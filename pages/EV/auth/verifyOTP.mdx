# verifyOTP API

## Description

The `verifyOTP` API endpoint is used to verify the OTP (One-Time Password) sent to a user's phone number during the authentication process.

## Endpoint Information

- **URL**: `http://localhost:3000/v1/auth/verify_otp`
- **Method**: `POST`
- **Authentication**: None

## Request Body

The request body must be in JSON format and include the following fields:

| Field   | Type   | Description                   |
|---------|--------|-------------------------------|
| phone   | string | The user's phone number       |
| otp     | string | The OTP sent to the phone     |

### Example

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

## Usage

To use this API, send a POST request to the endpoint URL with the appropriate JSON body. This will verify the OTP for the provided phone number. Below is an example of how to structure the request:

```http
POST http://localhost:3000/v1/auth/verify_otp
Content-Type: application/json

{
  "phone": "+919812940706",
  "otp": "815897"
}
```
