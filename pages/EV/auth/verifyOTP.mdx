# verifyOTP API

## Overview

The `verifyOTP` API is used to verify the OTP sent to a user's phone number.

## HTTP Request

### Method

`POST`

### URL

`http://localhost:3000/v1/auth/verify_otp`

## Request

### Headers

None

### Body

The request body must be in JSON format and include the following fields:

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

- **phone**: A string representing the user's phone number in an international format.
- **otp**: A string representing the one-time password (OTP) sent to the user.

## Authentication

No authentication is required to access this endpoint. 

## Response

The API's response will inform you whether the OTP verification was successful or not. The structure of the response will depend on the server's implementation.
