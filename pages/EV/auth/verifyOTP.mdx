# verifyOTP API

The `verifyOTP` API allows you to verify a One-Time Password (OTP) sent to a user's phone number.

## HTTP Request

### POST

**URL:** `http://localhost:3000/v1/auth/verify_otp`

**Auth:** None required

## Request Body

The request body should be sent in JSON format and contain the following fields:

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

| Field   | Type   | Description                            |
|---------|--------|----------------------------------------|
| `phone` | String | The phone number to which the OTP was sent. |
| `otp`   | String | The One-Time Password to be verified.      |

## Example

### Request

```json
POST http://localhost:3000/v1/auth/verify_otp
Content-Type: application/json

{
  "phone": "+919812940706",
  "otp": "815897"
}
```

## Response

The response structure will vary depending on the implementation. Typically, the server will return a status indicating whether the OTP was successfully verified or not.

Make sure to handle different response statuses to ensure robustness of your application.
