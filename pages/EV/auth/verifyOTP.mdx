## API Name: verifyOTP

This API endpoint is used to verify a one-time password (OTP).

### Endpoint

- **URL:** `http://localhost:3000/v1/auth/verify_otp`
- **Method:** `POST`
- **Authentication:** None required

### Request Body

The request body should be in JSON format and contain the following fields:

```json
{
  "phone": "+919812940706",
  "otp": "815897"
}
```

- **phone**: The phone number for which the OTP was generated. *(string, required)*
- **otp**: The OTP to be verified. *(string, required)*

### Sequence

- **Type:** HTTP
- **Sequence:** 2
