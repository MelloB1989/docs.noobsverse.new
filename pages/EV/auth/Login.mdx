# Login API

The Login API is used to authenticate a user by their phone number.

## HTTP Request

**POST** `https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login`

## Request Headers

| Header                  | Value                                | Description                          |
|-------------------------|--------------------------------------|--------------------------------------|
| `X-Karma-Admin-Auth`    | ajbkbakweiuy387yeuqqwfahdjhsabd      | Static token for initial authentication |

## Request Body

The request body should be sent in JSON format.

```json
{
  "phone": "+919812940706"
}
```

- `phone` (string): The phone number of the user.

## Authentication

This endpoint does not require any authentication tokens in the header aside from the static `X-Karma-Admin-Auth`. There is no additional authentication method required.
