## Overview

The Login API allows users to authenticate by providing their phone number. This API does not require any authentication headers, and the request body should be in JSON format.

## Endpoint

**POST** `https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login`

## Headers

| Header Name           | Value                                       |
| --------------------- | ------------------------------------------- |
| X-Karma-Admin-Auth    | ajbkbakweiuy387yeuqqwfahdjhsabd             |

## Request Body

The request body should be in JSON format:

```json
{
  "phone": "+919812940706"
}
```

## Example Request

```http
POST /v1/auth/login HTTP/1.1
Host: ev-backend.app.k8s.coffeecodes.in
Content-Type: application/json
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd

{
  "phone": "+919812940706"
}
```

## Response

The API will return a JSON response. The exact structure and details of the response will depend on the implementation of the backend server. Usually, it includes authentication tokens and user information upon successful login.
