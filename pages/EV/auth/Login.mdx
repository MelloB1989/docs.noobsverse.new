# Login API

## Overview

The Login API allows users to authenticate into the system by providing their phone number. This is the first step to access protected resources.

## HTTP Method

`POST`

## Endpoint

```plaintext
https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login
```

## Headers

| Key                   | Value                                      |
|-----------------------|--------------------------------------------|
| `X-Karma-Admin-Auth`  | ajbkbakweiuy387yeuqqwfahdjhsabd            |

## Request Body

The request body must be in JSON format.

```json
{
  "phone": "+919812940706"
}
```

## Authentication

No authentication is required for this endpoint.
