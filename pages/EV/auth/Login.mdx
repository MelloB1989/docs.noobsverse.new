# Login API

## Overview

The Login API is used to authenticate users via their phone numbers. This is an unauthenticated endpoint meaning you do not need a prior token to access it.

## Endpoint

**POST** `https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login`

## Headers

| Header                 | Value                                      | Required |
|------------------------|--------------------------------------------|----------|
| `X-Karma-Admin-Auth`   | ajbkbakweiuy387yeuqqwfahdjhsabd            | Yes      |

## Request Body

The request body must be in JSON format and includes the user's phone number.

### JSON Payload

```json
{
  "phone": "+919812940706"
}
```

## Example

### Curl Command

```bash
curl -X POST https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login \
     -H "X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd" \
     -H "Content-Type: application/json" \
     -d '{"phone": "+919812940706"}'
```

## Usage Notes

- Ensure the phone number is passed in the correct format.
- Provided `X-Karma-Admin-Auth` header is necessary for the request to be processed.
