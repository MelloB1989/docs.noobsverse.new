## Login API

The Login API allows users to authenticate using their phone number.

### Endpoint

**POST** `https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login`

### Headers

```json
{
  "X-Karma-Admin-Auth": "ajbkbakweiuy387yeuqqwfahdjhsabd"
}
```

### Request Body

The request body must be in JSON format.

```json
{
  "phone": "+919812940706"
}
```

### Authentication

No authentication is required for this endpoint.
