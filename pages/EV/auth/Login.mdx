# Login API

The Login API is used to authenticate a user using their phone number. Below are the detailed specifications for making a request to this API.

## Endpoint

- **Method**: `POST`
- **URL**: `https://ev-backend.app.k8s.coffeecodes.in/v1/auth/login`
- **Authorization**: None required

## Headers

Include the following headers in your request:

| Header Name            | Value                               |
|------------------------|-------------------------------------|
| X-Karma-Admin-Auth     | ajbkbakweiuy387yeuqqwfahdjhsabd     |

## Request Body

The request body should be in JSON format with the following structure:

```json
{
  "phone": "+919812940706"
}
```

Ensure that the phone number is in the correct format as shown above.

## Response

Upon successful login, the server will respond with a status indicating the result of the authentication process. The exact response JSON structure and status codes should be referred to in the full API documentation.
