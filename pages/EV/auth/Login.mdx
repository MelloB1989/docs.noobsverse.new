## Login API

The Login API allows users to authenticate by providing their phone number. Below are the detailed specifications for making a request to this API.

### HTTP Request

**Method**: POST

**URL**: `http://127.0.0.1:3000/v1/auth/login`

### Authorization

No authentication is required to access this endpoint.

### Headers

Include the following header in your request:

- `X-Karma-Admin-Auth`: `ajbkbakweiuy387yeuqqwfahdjhsabd`

### Request Body

The request body should be in JSON format and include the following field:

```json
{
  "phone": "+919812940706"
}
```

### Example Request

```json
POST /v1/auth/login HTTP/1.1
Host: 127.0.0.1:3000
Content-Type: application/json
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd

{
  "phone": "+919812940706"
}
```

Make sure to replace `+919812940706` with the actual phone number of the user trying to log in.
