# Signup API

## Description

The Signup API allows users to create a new account by providing their personal details and authentication information.

## Endpoint

`POST http://localhost:3000/v1/auth/signup`

## Request Format

- **URL:** `http://localhost:3000/v1/auth/signup`
- **Method:** `POST`
- **Authentication:** None required
- **Content-Type:** `application/json`

## Request Body

The request body should be in JSON format and include the following fields:

```json
{
  "phone": "+919812940706",
  "otp": "316892",
  "email": "mellob.mailer@gmail.com",
  "first_name": "Kartik",
  "last_name": "Deshmukh",
  "date_of_birth": "27/11/2004",
  "password": "password",
  "type": "student",
  "employee_id": "",
  "company": "",
  "college": "AU",
  "student_id": "34"
}
```

### Field Details

- **phone** (string): The user's phone number. Example: `"+919812940706"`
- **otp** (string): The one-time password sent to the user's phone. Example: `"316892"`
- **email** (string): The user's email address. Example: `"mellob.mailer@gmail.com"`
- **first_name** (string): The user's first name. Example: `"Kartik"`
- **last_name** (string): The user's last name. Example: `"Deshmukh"`
- **date_of_birth** (string): The user's date of birth in `DD/MM/YYYY` format. Example: `"27/11/2004"`
- **password** (string): The user's password. Example: `"password"`
- **type** (string): The type of user. Possible values: `"student"`, `"employee"`
  - If **type** is `"student"`, the following fields are required:
    - **college** (string): Name of the college. Example: `"AU"`
    - **student_id** (string): The student's ID. Example: `"34"`
  - If **type** is `"employee"`, the following fields are required:
    - **employee_id** (string): The employee's ID.
    - **company** (string): Name of the company.

## Response

On successful signup, the API will respond with a status code of `201` and a message indicating that the user has been created successfully.

---

If there are any issues or errors, an appropriate error message and status code will be returned.
