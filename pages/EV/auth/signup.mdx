# Signup API

## Meta
- **Name**: signup
- **Type**: http
- **Sequence**: 3

## Post Request

### URL
`http://localhost:3000/v1/auth/signup`

### Body
The body of the POST request should be in JSON format.

### Authentication
No authentication is required for this endpoint.

## Request Body

### JSON Structure
```json
{
  "phone": "+919812940706",
  "otp": "316892",
  "email": "mellob.mailer@gmail.com",
  "first_name": "Kartik",
  "last_name": "Deshmukh",
  "date_of_birth": "27/11/2004",
  "password": "password",
  "type": "student",
  "employee_id": "",
  "company": "",
  "college": "AU",
  "student_id": "34"
}
```

### Fields

- **phone**: The phone number of the user. *(Example: "+919812940706")*
- **otp**: One-time password sent to the user's phone. *(Example: "316892")*
- **email**: The email address of the user. *(Example: "mellob.mailer@gmail.com")*
- **first_name**: The first name of the user. *(Example: "Kartik")*
- **last_name**: The last name of the user. *(Example: "Deshmukh")*
- **date_of_birth**: The date of birth of the user in DD/MM/YYYY format. *(Example: "27/11/2004")*
- **password**: The password for the user's account. *(Example: "password")*
- **type**: The type of user. *(Example: "student")*
- **employee_id**: The employee ID of the user, if applicable. *(Example: "")*
- **company**: The company of the user, if applicable. *(Example: "")*
- **college**: The college of the user. *(Example: "AU")*
- **student_id**: The student's ID of the user. *(Example: "34")*
