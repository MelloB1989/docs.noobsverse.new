# Signup API

The Signup API allows users to create a new account by providing their personal information and credentials. This API endpoint handles the registration process.

## Endpoint Overview

- **Name**: Signup
- **Type**: HTTP
- **Sequence**: 3

## HTTP Request

- **Method**: POST
- **URL**: `http://localhost:3000/v1/auth/signup`
- **Body Format**: JSON
- **Authentication**: None required

## Request Body

The following JSON payload should be included in the body of the request:

```json
{
  "phone": "+918266187862",
  "otp": "123456",
  "email": "manasveer6@gmail.com",
  "first_name": "Manas",
  "last_name": "Veer",
  "date_of_birth": "24/02/2005",
  "password": "passwordgobrr",
  "type": "student",
  "employee_id": "",
  "company": "",
  "college": "AU",
  "student_id": "62"
}
```

### Parameters

- **phone**: The user's phone number, including the country code (e.g., `+918266187862`).
- **otp**: The One Time Password (OTP) sent to the user's phone.
- **email**: The user's email address.
- **first_name**: The user's first name.
- **last_name**: The user's last name.
- **date_of_birth**: The user's date of birth, formatted as `DD/MM/YYYY`.
- **password**: The user's chosen password for the account.
- **type**: The type of user account. Possible value is `"student"`.
- **employee_id**: The employee ID, if applicable. Empty for students.
- **company**: The company name, if applicable. Empty for students.
- **college**: The user's college name (e.g., `"AU"` for Andhra University).
- **student_id**: The student's identification number.
