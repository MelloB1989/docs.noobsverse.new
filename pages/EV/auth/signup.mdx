# Signup API

The Signup API allows new users to register to the system by submitting their personal and authentication details.

## HTTP Method

**POST**

## Endpoint

```
http://localhost:3000/v1/auth/signup
```

## Authentication

This endpoint does not require authentication.

## Request Body

The body of the request should be in JSON format and include the following attributes:

```json
{
  "phone": "+919812940706",
  "otp": "316892",
  "email": "mellob.mailer@gmail.com",
  "first_name": "Kartik",
  "last_name": "Deshmukh",
  "date_of_birth": "27/11/2004",
  "password": "password",
  "type": "student",
  "employee_id": "",
  "company": "",
  "college": "AU",
  "student_id": "34"
}
```

### Attributes

- **phone** (string): The user's phone number, including country code.
- **otp** (string): The one-time password sent to the user's phone.
- **email** (string): The user's email address.
- **first_name** (string): The user's first name.
- **last_name** (string): The user's last name.
- **date_of_birth** (string): The user's date of birth in the format `dd/mm/yyyy`.
- **password** (string): The user's password.
- **type** (string): The type of user, can be `student`.
- **employee_id** (string): The employee ID if the user is an employee (can be left empty for non-employees).
- **company** (string): The company if the user is an employee (can be left empty for non-employees).
- **college** (string): The college name if the user is a student.
- **student_id** (string): The student ID if the user is a student.
