# Signup API

The Signup API allows users to register by providing their personal and contact information.

## POST Request

**URL**: `http://localhost:3000/v1/auth/signup`

**Body Format**: JSON

**Authentication**: None required

## Request Body

The following fields should be included in the JSON body of the POST request:

| Field          | Type   | Description                                | Example                    |
|----------------|--------|--------------------------------------------|----------------------------|
| `phone`        | string | User's phone number (with country code)    | `"+919812940706"`          |
| `otp`          | string | OTP received by the user                   | `"316892"`                 |
| `email`        | string | User's email address                       | `"mellob.mailer@gmail.com"`|
| `first_name`   | string | User's first name                          | `"Kartik"`                 |
| `last_name`    | string | User's last name                           | `"Deshmukh"`               |
| `date_of_birth`| string | User's date of birth (dd/mm/yyyy)          | `"27/11/2004"`             |
| `password`     | string | User's password                            | `"password"`               |
| `type`         | string | Type of user ("student" or other types)    | `"student"`                |
| `employee_id`  | string | Employee ID (if applicable)                | `""`                       |
| `company`      | string | Company name (if applicable)               | `""`                       |
| `college`      | string | College name (if applicable)               | `"AU"`                     |
| `student_id`   | string | Student ID (if applicable)                 | `"34"`                     |
