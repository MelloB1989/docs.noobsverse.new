# Signup API

## Endpoint
`POST http://localhost:3000/v1/auth/signup`

## Description
This endpoint allows a new user to sign up by providing their details.

## Request Body
The request body should be in JSON format and must include the following properties:

```json
{
  "phone": "+918266187862",
  "otp": "123456",
  "email": "manasveer6@gmail.com",
  "first_name": "Manas",
  "last_name": "Veer",
  "date_of_birth": "24/02/2005",
  "password": "passwordgobrr",
  "type": "student",
  "employee_id": "",
  "company": "",
  "college": "AU",
  "student_id": "62"
}
```

### Parameters

| Name          | Type   | Description                         |
|---------------|--------|-------------------------------------|
| phone         | string | User's phone number in international format |
| otp           | string | One-time password sent to user's phone |
| email         | string | User's email address                |
| first_name    | string | User's first name                   |
| last_name     | string | User's last name                    |
| date_of_birth | string | User's date of birth in DD/MM/YYYY format |
| password      | string | Password chosen by the user         |
| type          | string | User type (e.g., student)           |
| employee_id   | string | Employee ID (optional, if user is not a student) |
| company       | string | Company name (optional, if user is not a student) |
| college       | string | College name                        |
| student_id    | string | Student ID                          |

## Authentication
No authentication is required to access this endpoint.
