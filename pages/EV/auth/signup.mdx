# Signup API

## Endpoint

**POST** `http://localhost:3000/v1/auth/signup`

## Authentication

None required

## Request Body

The request body should be in JSON format and include the following fields:

| Field          | Type   | Description                        |
|----------------|--------|------------------------------------|
| phone          | string | User's phone number (e.g., "+919812940706") |
| otp            | string | One-time password (e.g., "316892") |
| email          | string | User's email address (e.g., "mellob.mailer@gmail.com") |
| first_name     | string | User's first name (e.g., "Kartik") |
| last_name      | string | User's last name (e.g., "Deshmukh") |
| date_of_birth  | string | User's date of birth (e.g., "27/11/2004") |
| password       | string | User's chosen password |
| type           | string | User type (e.g., "student") |
| employee_id    | string | Employee ID (if applicable, otherwise leave empty) |
| company        | string | Company name (if applicable, otherwise leave empty) |
| college        | string | College name (e.g., "AU") |
| student_id     | string | Student ID (e.g., "34") |

### Example

```json
{
  "phone": "+919812940706",
  "otp": "316892",
  "email": "mellob.mailer@gmail.com",
  "first_name": "Kartik",
  "last_name": "Deshmukh",
  "date_of_birth": "27/11/2004",
  "password": "password",
  "type": "student",
  "employee_id": "",
  "company": "",
  "college": "AU",
  "student_id": "34"
}
```
