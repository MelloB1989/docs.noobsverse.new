# Get User Subscription

Fetch the details of a user subscription from the API.

## Endpoint

- **URL:** `http://localhost:3000/v1/user_subscription/get`
- **Method:** GET
- **Authorization:** Bearer Token
- **Body Type:** JSON

```json

  "start_date": "27/11/2004",
  "end_date": "27/11/2004"
```
