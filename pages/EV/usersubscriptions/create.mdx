# Create User Subscription API

This API endpoint allows you to create a new user subscription.

## Endpoint Details

- **Type**: HTTP
- **Sequence**: 1
- **Method**: POST
- **URL**: `http://localhost:3000/v1/user_subscription/create`
- **Content Type**: JSON
- **Authentication**: Bearer Token

## Request Body

The request body should be in JSON format and must contain the following fields:

- `subscription_id` (string): The unique identifier for the subscription.
- `start_date` (string): The start date of the subscription in `DD/MM/YYYY` format.
- `end_date` (string): The end date of the subscription in `DD/MM/YYYY` format.

### Example JSON Body

```json
{
  "subscription_id": "qcxt0ky6xg",
  "start_date": "27/11/2004",
  "end_date": "27/11/2004"
}
```

## Response

Upon successful creation of a user subscription, the server will respond with a status code indicating success and any relevant information about the created subscription.

Please ensure that all fields in the request body are correctly formatted and valid to avoid errors.

## Summary

The Create User Subscription API is a straightforward POST request that requires authentication via a bearer token and a JSON-formatted body specifying the subscription details.
