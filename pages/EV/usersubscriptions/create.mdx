# User Subscription - Create API

The **Create API** allows you to create a new user subscription. This document details the necessary endpoints, authentication, headers, and body requirements to make a request.

## Endpoint

- **URL**: `http://localhost:3000/v1/user_subscription/create`
- **Method**: `POST`

## Request Body

The request body must be in JSON format and include the following fields:

```json
{
  "subscription_id": "qcxt0ky6xg",
  "start_date": "27/11/2004",
  "end_date": "27/11/2004"
}
```

### Body Parameters

- **subscription_id**: A unique identifier for the subscription. *(String)*
- **start_date**: The start date of the subscription in the format `dd/mm/yyyy`. *(String)*
- **end_date**: The end date of the subscription in the format `dd/mm/yyyy`. *(String)*
