# API Documentation: Create User Subscription

This document provides details on how to use the `create` API endpoint to add a new user subscription.

## Endpoint Type

- **Type**: HTTP
- **Method Sequence**: 1

## HTTP Method

### POST

- **URL**: `http://localhost:3000/v1/user_subscription/create`
- **Body Format**: JSON
- **Authentication**: Bearer token

## Authentication

To access this endpoint, a valid Bearer token must be provided in the Authorization header.

## Request Body

The body of the request must be in JSON format and include the following fields:

```json
{
  "subscription_id": "qcxt0ky6xg",
  "start_date": "27/11/2004",
  "end_date": "27/11/2004"
}
```

- **subscription_id**: A unique identifier for the subscription.
- **start_date**: The date when the subscription starts, in `dd/mm/yyyy` format.
- **end_date**: The date when the subscription ends, in `dd/mm/yyyy` format.

## Notes

Ensure that the Bearer token is kept secure and replaced with a valid token as necessary. The provided token is for demonstration purposes only and may not be valid for actual API requests.
