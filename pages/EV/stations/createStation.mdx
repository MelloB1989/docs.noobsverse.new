# createStation API

## Description
The `createStation` API allows the creation of new stations with specified details such as name, location, capacity, and current capacity.

## Endpoint

### HTTP Method: POST
- **URL**: `http://localhost:3000/v1/stations/create`

### Authentication
- **None**

### Headers
- **X-Karma-Admin-Auth**: `CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab`
