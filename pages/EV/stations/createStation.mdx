## Create Station API

### Endpoint
**POST** `http://localhost:3000/v1/stations/create`

### Authentication
None required.

### Headers
| Key                    | Value                                                                                                       |
|------------------------|-------------------------------------------------------------------------------------------------------------|
| `X-Karma-Admin-Auth`   | `CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab`                             |

### Request Body
The request body should be a JSON array containing station details. Each station object should have the following fields:

| Field               | Type    | Description                        |
|---------------------|---------|------------------------------------|
| `name`              | string  | The name of the station            |
| `location_latitude` | string  | The latitude of the station's location |
| `location_longitude`| string  | The longitude of the station's location|
| `capacity`          | number  | The capacity of the station        |
| `current_capacity`  | number  | The current capacity of the station |
