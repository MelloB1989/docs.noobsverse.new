## Description

The `createStation` API allows users to create multiple station records in one request. Each station information must include:

- `name` (string): The name of the station.
- `location_latitude` (string): The latitude of the station's location.
- `location_longitude` (string): The longitude of the station's location.
- `capacity` (integer): The total capacity the station can hold.
- `current_capacity` (integer): The current number of occupants or units at the station.

Ensure that all data points are provided correctly for successful creation of the station records.
