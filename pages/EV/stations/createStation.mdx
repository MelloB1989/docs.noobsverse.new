# Create Station API Documentation

## Overview

The `createStation` API allows you to create new stations by sending a set of details for each station. This endpoint accepts a JSON payload containing information about each station.

## Endpoint

- **URL**: `http://localhost:3000/v1/stations/create`
- **Method**: `POST`
- **Authentication**: None required

## Headers

To authorize the request, include the following header:

## Request Body

The request body should be in JSON format, consisting of an array of station objects with the following structure:

```json
[
  {
    "name": "station_name",
    "location_latitude": "latitude_value",
    "location_longitude": "longitude_value",
    "capacity": capacity_value,
    "current_capacity": current_capacity_value
  }
]
```

### Parameters

- **name**: `string`  
  The name of the station.

- **location_latitude**: `string`  
  The latitude of the station's location.

- **location_longitude**: `string`  
  The longitude of the station's location.

- **capacity**: `integer`  
  The total capacity of the station.

- **current_capacity**: `integer`  
  The current usage capacity of the station.
