# getStations API

The `getStations` endpoint is used to retrieve a list of stations. This document explains how to interact with this endpoint.

## Meta Information

- **Name**: getStations
- **Type**: HTTP
- **Sequence**: 2

## HTTP GET Request

- **URL**: `http://localhost:3000/v1/stations/get`
- **Body**: None
- **Authentication Required**: None

## Headers

The request must include the following header:

- **X-Karma-Admin-Auth**: `ajbkbakweiuy387yeuqqwfahdjhsabd`

## Example Request

```http
GET /v1/stations/get HTTP/1.1
Host: localhost:3000
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Response

The response will contain the list of stations in JSON format.

```json
{
  "stations": [
    {
      "id": "station1",
      "name": "Station 1"
    },
    {
      "id": "station2",
      "name": "Station 2"
    }
  ]
}
```

## Notes

- Ensure that the `X-Karma-Admin-Auth` header is included with the correct value to successfully authenticate the request.
