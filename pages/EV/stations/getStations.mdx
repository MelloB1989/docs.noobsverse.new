# getStations API

## Endpoint

**GET** `http://localhost:3000/v1/stations/get`

## Request

- **Body**: None
- **Auth**: None

| Header                 | Value                              |

## Description

The `getStations` API endpoint retrieves the list of stations. This endpoint does not require any request body or authentication. However, a specific admin authorization header (`X-Karma-Admin-Auth`) must be included in the request.
