## Get Stations API

The `getStations` endpoint allows you to retrieve a list of stations.

### Endpoint

**GET** `http://localhost:3000/v1/stations/get`

### Authentication

No authentication is required for this endpoint.

### Headers

Include the following header to authorize your request:

- `X-Karma-Admin-Auth`: `ajbkbakweiuy387yeuqqwfahdjhsabd`

### Request Body

This endpoint does not require a request body.

### Response

On success, the endpoint will return a list of stations.

### Example Request

```http
GET http://localhost:3000/v1/stations/get
```

### Example Response

```json
[
  {
    "id": "station1",
    "name": "First Station",
    "location": "Location 1"
  },
  {
    "id": "station2",
    "name": "Second Station",
    "location": "Location 2"
  }
]
```

### Status Codes

- `200 OK`: The request was successful, and the list of stations is returned.
- `400 Bad Request`: The request could not be understood or was missing required parameters.
- `401 Unauthorized`: Authentication failed or was not provided.
- `500 Internal Server Error`: An error occurred on the server.

### Notes

Make sure to replace the placeholder in the `X-Karma-Admin-Auth` header with the correct authentication token.

For further information, please contact the API support team.
