# getStations API

The `getStations` API is an HTTP endpoint designed to retrieve station data. Below, you'll find detailed information about the endpoint, including its URL, authentication requirements, and headers.

## Endpoint Information

- **Type:** HTTP
- **Sequence:** 2

## HTTP GET Request

To access the station data, make a `GET` request to the following URL:

### URL

```
http://localhost:3000/v1/stations/get
```

### Request Body

No request body is required for this endpoint.

### Authentication

No authentication is required to access this endpoint.

### Headers

To successfully call the `getStations` API, include the following header in your request:

- `X-Karma-Admin-Auth`: Use the following value for the header:

```
ajbkbakweiuy387yeuqqwfahdjhsabd
```

Make sure the header is correctly included to ensure a successful API call.
