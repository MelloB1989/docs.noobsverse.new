# getStations API

## Overview

The `getStations` API endpoint retrieves a list of stations from the server.

- **Type**: `HTTP`
- **Sequence**: `2`

## HTTP GET Request

### URL

```plaintext
http://localhost:3000/v1/stations/get
```

### Headers

The following headers are required when making a request to the `getStations` API:

| Header                 | Value                                 |
| ---------------------- | ------------------------------------- |
| `X-Karma-Admin-Auth`   | ajbkbakweiuy387yeuqqwfahdjhsabd       |

### Request Body

None.

### Authentication

None.

## Example Request

```http
GET /v1/stations/get HTTP/1.1
Host: localhost:3000
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Response

The API will return a JSON response containing the list of stations.

## Conclusion

The `getStations` API provides a straightforward way to retrieve station data without requiring any authentication. Ensure to include the necessary headers as specified to successfully access the endpoint.
