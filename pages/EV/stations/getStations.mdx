# getStations API

The `getStations` API retrieves a list of stations. This document provides the necessary details to use this API effectively.

## Endpoint

### GET Request

- **URL**: `http://localhost:3000/v1/stations/get`
- **Method**: `GET`
- **Body**: None
- **Authentication**: None

### Headers

To successfully call this API, include the following header:

- **X-Karma-Admin-Auth**: `ajbkbakweiuy387yeuqqwfahdjhsabd`

## Example Request

```http
GET http://localhost:3000/v1/stations/get
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Responses

### Success

- **Status Code**: `200 OK`
- **Description**: Successfully retrieved the list of stations.
- **Body**: A JSON array containing station details.

### Error

- **Status Code**: `401 Unauthorized`
- **Description**: Headers are missing or the authentication token is invalid.
- **Body**: An error message explaining the issue.

---

Ensure you replace the placeholder authentication token with a valid one provided to you.
