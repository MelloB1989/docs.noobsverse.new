# API Documentation: `getClosestStation`

## Endpoint
**GET** `https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2`

## Description
Fetches the closest electric vehicle charging stations based on provided latitude, longitude, and limit.

## Parameters

### Query Parameters
- `lat`: Latitude (e.g., `40`)
- `lon`: Longitude (e.g., `-70`)
- `limit`: Number of nearby stations to retrieve (e.g., `2`)

## Headers
- `X-Karma-Admin-Auth`: `ajbkbakweiuy387yeuqqwfahdjhsabd`

### Example Request
```http
GET https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

No request body is required for this endpoint. No authentication is required beyond the specified header.
