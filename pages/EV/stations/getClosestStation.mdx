# Get Closest Station

This API retrieves the closest electric vehicle charging stations based on the provided latitude and longitude.

## HTTP Method

`GET`

## Endpoint

`https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby`

## Query Parameters

- **lat**: Latitude for the location around which to search (e.g., `40`).
- **lon**: Longitude for the location around which to search (e.g., `-70`).
- **limit**: Number of stations to return (e.g., `2`).

## Request Headers

- **X-Karma-Admin-Auth**: `ajbkbakweiuy387yeuqqwfahdjhsabd` (Authentication token required for accessing the API).

## Request Body

None

## Authentication

No additional authentication is required beyond the header provided.

## Example Request

```http
GET /v1/stations/get_nearby?lat=40&lon=-70&limit=2 HTTP/1.1
Host: ev-backend.app.k8s.coffeecodes.in
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```
