# API Documentation: getClosestStation

The `getClosestStation` API retrieves the nearest stations based on the provided latitude and longitude.

## Endpoint

**GET** `https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2`

## Query Parameters

- **lat**: Latitude of the location (e.g., 40)
- **lon**: Longitude of the location (e.g., -70)
- **limit**: Maximum number of stations to return (e.g., 2)

## Headers

- **X-Karma-Admin-Auth**: `ajbkbakweiuy387yeuqqwfahdjhsabd`

## Request Example

```http
GET /v1/stations/get_nearby?lat=40&lon=-70&limit=2 HTTP/1.1
Host: ev-backend.app.k8s.coffeecodes.in
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Notes

- No request body or additional authorization is required beyond the provided header.
- Ensure the `X-Karma-Admin-Auth` header is included for successful authentication.
