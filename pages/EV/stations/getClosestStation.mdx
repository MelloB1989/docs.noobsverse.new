# Get Closest Station API

## Description
The `getClosestStation` API retrieves nearby charging stations based on provided latitude and longitude coordinates. You can limit the number of stations returned by specifying the `limit` parameter.

## HTTP Request

### GET
`https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby`

### Query Parameters
- `lat`: Latitude coordinate (example: `40`)
- `lon`: Longitude coordinate (example: `-70`)
- `limit`: Number of stations to return (example: `2`)

### Headers
- `X-Karma-Admin-Auth`: `ajbkbakweiuy387yeuqqwfahdjhsabd`

## Example Request

```http
GET /v1/stations/get_nearby?lat=40&lon=-70&limit=2 HTTP/1.1
Host: ev-backend.app.k8s.coffeecodes.in
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Response
A successful response contains a list of nearby charging stations. Each station includes details such as station ID, name, location coordinates, and other relevant information.

## Notes
- Authentication is needed using the `X-Karma-Admin-Auth` header.
- Ensure the query parameters are formatted correctly to receive accurate results.
