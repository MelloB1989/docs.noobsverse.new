# getClosestStation API

The `getClosestStation` API endpoint is used to retrieve the nearest stations based on the provided latitude and longitude, with an optional limit on the number of results.

## HTTP Request

**GET** `https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby`

## Query Parameters

| Name  | Type | Description                                        |
|-------|------|----------------------------------------------------|
| lat   | float| Latitude coordinate to search nearby stations.    |
| lon   | float| Longitude coordinate to search nearby stations.   |
| limit | int  | Maximum number of stations to return (optional).  |

## Default Query Example

```text
lat=40&lon=-70&limit=2
```

## Request Headers

| Name                 | Description                             |
|----------------------|-----------------------------------------|
| X-Karma-Admin-Auth   | Authentication token for admin access.  |

## Sample Request

```http
GET https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Response

The API response will contain details about the closest stations based on the provided latitude and longitude parameters. The structure of the response will depend on the backend implementation and may vary.
