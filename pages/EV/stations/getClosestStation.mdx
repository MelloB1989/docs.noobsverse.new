# API Documentation: getClosestStation

## Endpoint Details

**Method**: GET

**URL**: `https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2`

## Request Parameters

### Query Parameters
| Parameter | Type | Description           | Example       |
|-----------|------|-----------------------|---------------|
| `lat`     | float| Latitude of the area  | `40`          |
| `lon`     | float| Longitude of the area | `-70`         |
| `limit`   | int  | Number of stations to be retrieved | `2`         |

## Request Headers

| Header                   | Value                                 | Description                       |
|--------------------------|---------------------------------------|-----------------------------------|
| `X-Karma-Admin-Auth`     | `ajbkbakweiuy387yeuqqwfahdjhsabd`     | Authentication token for the API  |

## Authentication

No additional authentication is required beyond the `X-Karma-Admin-Auth` header provided above.

## Request Body

None.

## Response

The API will return the closest electric vehicle stations based on the provided latitude and longitude, limited to the specified number of stations. The response content and structure are not detailed in this documentation.

## Example Request

```http
GET https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Notes

- Ensure the `lat` and `lon` parameters are valid coordinates.
- The `limit` parameter controls the number of stations returned and should be a positive integer.

This endpoint is designed to quickly retrieve nearby stations based on geographic coordinates provided.
