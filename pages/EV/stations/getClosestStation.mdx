# getClosestStation API

The `getClosestStation` API allows you to fetch the nearest charging stations based on latitude and longitude coordinates.

## Request

### HTTP Method

- `GET`

### URL

```
https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2
```

### Headers

- `X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd`

### Query Parameters

| Parameter | Type   | Required | Description                           |
|-----------|--------|----------|---------------------------------------|
| lat       | float  | Yes      | The latitude for the search location. |
| lon       | float  | Yes      | The longitude for the search location. |
| limit     | int    | No       | The number of results to return. Default is 2. |

## Example Request

```
GET https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2
Headers:
  X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Authentication

This API does not require additional user authentication, but it does require a specific header for admin authorization.

## Response

On success, the API will return a JSON object containing the nearest charging stations based on the given latitude and longitude.

## Notes

- Ensure that the `X-Karma-Admin-Auth` header is included in the request to authorize API access.
- The default limit for the number of charging stations returned is 2, unless specified otherwise.
