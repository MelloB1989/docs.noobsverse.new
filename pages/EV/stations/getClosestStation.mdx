## getClosestStation API

The `getClosestStation` API is an HTTP-based service designed to fetch the nearest stations based on geographic coordinates. The service does not require authentication and provides a straightforward query interface to obtain location-specific data.

### Endpoint

- **HTTP Method:** GET
- **URL:** `https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby`

### Query Parameters

- **lat**: The latitude coordinate around which to search for stations.
- **lon**: The longitude coordinate around which to search for stations.
- **limit**: The maximum number of stations to return.

### Required Headers

To access the `getClosestStation` API, include the following header in your request:

- **X-Karma-Admin-Auth**: `ajbkbakweiuy387yeuqqwfahdjhsabd`

### Sample Request

```http
GET /v1/stations/get_nearby?lat=40&lon=-70&limit=2 HTTP/1.1
Host: ev-backend.app.k8s.coffeecodes.in
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

### Response

This API returns a list of nearby stations based on the provided latitude and longitude. The response includes key information about each station such as its ID, name, location, and other relevant details.

### Notes

- Ensure that the latitude (`lat`) and longitude (`lon`) are provided as query parameters.
- The `limit` parameter restricts the number of stations returned by the API.
- The X-Karma-Admin-Auth header is mandatory for the request to be processed successfully.
