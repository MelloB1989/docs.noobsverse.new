## Overview

The **getClosestStation** API is a part of the HTTP-based services designed to retrieve information about nearby stations based on specified latitude and longitude. This endpoint offers a simple yet effective means of fetching up to a set number of nearby stations.

### Endpoint

- **URL**: `https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby`
- **Method**: GET
- **Auth**: None required

### Parameters

The API uses query parameters to determine the search criteria for nearby stations:

- **lat**: Latitude of the location (e.g., `40`)
- **lon**: Longitude of the location (e.g., `-70`)
- **limit**: The maximum number of nearby stations to return (e.g., `2`)

Example:

```
https://ev-backend.app.k8s.coffeecodes.in/v1/stations/get_nearby?lat=40&lon=-70&limit=2
```

### Headers

To access the API, include the following header:

- **X-Karma-Admin-Auth**: Authentication token. Example: `ajbkbakweiuy387yeuqqwfahdjhsabd`

### Response

The API returns a list of the closest stations, limited by the `limit` parameter in the query.
