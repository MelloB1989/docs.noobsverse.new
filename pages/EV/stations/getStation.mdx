# getStation API

## Overview

The `getStation` API allows clients to retrieve information about a specific station. This API endpoint is a part of the HTTP type services.

## HTTP Method

- **GET**

## Endpoint

```
http://localhost:3000/v1/stations/get/wxyoqxff83
```

## Authentication

- **None required**

## Headers

To access the `getStation` API, the following header is required:

- `X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd`

## Request Body

- **None**

## Usage Example

```http
GET http://localhost:3000/v1/stations/get/wxyoqxff83
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Sequence

- **3**

## Notes

Make sure to replace the `X-Karma-Admin-Auth` value with the appropriate authentication token if you're using a different environment or updating the token for security purposes.
