# getStation API

The `getStation` API is an HTTP endpoint that allows you to retrieve information about a specific station. Below is the detailed documentation for using this API.

## Meta Information

- **Name:** getStation
- **Type:** HTTP
- **Sequence:** 3

## Endpoint

- **Method:** GET
- **URL:** `http://localhost:3000/v1/stations/get/wxyoqxff83`

## Request Details

- **Request Body:** None
- **Authentication:** None required

## Headers

You need to include the following header in your request:

- **X-Karma-Admin-Auth:** `ajbkbakweiuy387yeuqqwfahdjhsabd`

## Response

The response from this endpoint will include information related to the requested station. Details about the response structure are not provided in this documentation and should be referred to in related backend API documentation or examples.

Ensure you have the appropriate header set as shown to successfully call this API endpoint.
