# getStation API

The `getStation` API is used to fetch details of a specific station. Below are the details for the API endpoint and its usage.

## HTTP Request

### GET

```http
GET http://localhost:3000/v1/stations/get/wxyoqxff83
```

## Request Headers

| Header                 | Value                                    | Description                             |
|------------------------|------------------------------------------|-----------------------------------------|
| X-Karma-Admin-Auth     | ajbkbakweiuy387yeuqqwfahdjhsabd          | Authentication token for admin access   |

## Request Body

No body parameters are required for this request.

## Authentication

No additional authentication is required other than the `X-Karma-Admin-Auth` header.

## Example Request

```http
GET http://localhost:3000/v1/stations/get/wxyoqxff83
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Notes

- Make sure to replace the `X-Karma-Admin-Auth` value with a valid authentication token.
- Ensure the endpoint URL is correct and accessible from your server environment.
