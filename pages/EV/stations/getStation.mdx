# getStation API

The **getStation** API is used to retrieve information about a station. This endpoint is a part of the API services and is available via a GET request.

## HTTP Request

### URL

`GET http://localhost:3000/v1/stations/get/wxyoqxff83`

### Method

GET

### Body

None

### Authentication

None

### Headers

| Header Name           | Header Value                            |
|-----------------------|------------------------------------------|
| X-Karma-Admin-Auth    | ajbkbakweiuy387yeuqqwfahdjhsabd          |

## Example Request

```plaintext
GET http://localhost:3000/v1/stations/get/wxyoqxff83
X-Karma-Admin-Auth: ajbkbakweiuy387yeuqqwfahdjhsabd
```

## Response

The response will contain the details of the specified station. The exact structure of the response will depend on the station's data schema.
