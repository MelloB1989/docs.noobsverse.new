# getTimeTraveled API

## Overview
The `getTimeTraveled` API endpoint allows users to get the time traveled within a specified date range.

## HTTP Request
### Method: GET
### URL
```
http://localhost:3000/v1/user/time_travelled
```

### Query Parameters
| Name       | Type   | Description                      | Example          |
|------------|--------|----------------------------------|------------------|
| start_date | string | The start date of the period     | 2024-09-01       |
| end_date   | string | The end date of the period       | 2024-09-10       |
