# getTimeTraveled API

This document provides detailed information about the `getTimeTraveled` API.

## Endpoint

### URL
`GET http://localhost:3000/v1/user/time_travelled`

### Method
`GET`

## Query Parameters

- `start_date`: The start date for the time travel query in `YYYY-MM-DD` format. Example: `2024-09-01`
- `end_date`: The end date for the time travel query in `YYYY-MM-DD` format. Example: `2024-09-10`
