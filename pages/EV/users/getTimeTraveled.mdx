# getTimeTraveled API

The `getTimeTraveled` API provides information about the time a user has traveled within a specified date range. 

## Endpoint

### HTTP Method: GET

- **URL**: `http://localhost:3000/v1/user/time_travelled`

### Query Parameters

- `start_date`: Date from which to start calculating the time traveled. Format: `YYYY-MM-DD`.
  - **Example**: `2024-09-01`

- `end_date`: Date until which to calculate the time traveled. Format: `YYYY-MM-DD`.
  - **Example**: `2024-09-10`
