## Overview

The `getTimeTraveled` API endpoint fetches the time traveled by a user between specified start and end dates. This endpoint requires authentication via a bearer token. 

## Endpoint Details

- **Method**: `GET`
- **URL**: `http://localhost:3000/v1/user/time_travelled`

## Query Parameters

The API requires two query parameters representing the time period for which the travel time is to be fetched:

- **start_date**: The start date of the period (Format: `YYYY-MM-DD`).
  - Example: `2024-09-01`

- **end_date**: The end date of the period (Format: `YYYY-MM-DD`).
  - Example: `2024-09-10`
