# unfollowUser API

## Overview

The `unfollowUser` API allows a client to unfollow a user by making an HTTP GET request. Authentication is required via a Bearer token.

- **Type:** HTTP
- **Sequence:** 6

## Endpoint

### GET Request

- **URL:** `http://localhost:3000/v1/user/unfollow/{user_id}`

  Replace `{user_id}` with the actual user ID to unfollow (e.g., `-sx7zvg7n1`).

- **Body:** None

- **Authentication:** Bearer token is required.
