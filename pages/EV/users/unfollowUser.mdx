# unfollowUser API

## Overview

The `unfollowUser` API allows authenticated users to unfollow another user.

## Endpoint

### GET Request

**URL:** `http://localhost:3000/v1/user/unfollow/:userId`

Replace `:userId` with the actual user ID of the user you wish to unfollow.

**Body:** None

**Authentication:** Bearer token required
