# searchByFirstName API

The `searchByFirstName` API allows you to search for users by their first name. Below are the details of the API endpoint and how to use it.

## HTTP Method

GET

## Endpoint

```plaintext
http://localhost:3000/v1/user/search/manas
```

## Authentication

This API uses Bearer Token authentication. You must include a valid token in the Authorization header of your request.

### Bearer Token

```plaintext
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

## Request Body

No request body is needed for this endpoint.

## Example Request

```http
GET /v1/user/search/manas HTTP/1.1
Host: localhost:3000
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

## Response

The API will return a JSON response with user details matching the first name "manas".

## Errors

Ensure that you provide a valid Bearer Token; otherwise, the API will respond with an authentication error. If the requested user cannot be found or an internal server error occurs, appropriate error messages will be returned.
