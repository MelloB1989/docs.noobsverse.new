## Update User API

An API endpoint to update user details.

### Endpoint Details

- **Name:** `updateUser`
- **Type:** `HTTP`
- **Sequence:** `4`

### HTTP Request

- **Method:** `POST`
- **URL:** `http://localhost:3000/v1/user/update`
- **Body Type:** `JSON`
