# followUser API

## Overview

The `followUser` API allows a user to follow another user. This API requires a bearer token for authentication.

## Request

### GET

- **URL:** `http://localhost:3000/v1/user/follow/-sx7zvg7n1`
- **Body:** None
- **Authentication:** Bearer Token
