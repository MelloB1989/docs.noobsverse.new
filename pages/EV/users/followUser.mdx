# followUser API

The `followUser` API allows a client to follow a specific user. This document provides details on how to authenticate and make a request to the endpoint.

## Metadata

- **Name:** followUser
- **Type:** HTTP
- **Sequence:** 5

## Endpoint

### HTTP GET Request

- **URL:** `http://localhost:3000/v1/user/follow/-sx7zvg7n1`
- **Body:** None
- **Authentication Required:** Yes, Bearer Token

