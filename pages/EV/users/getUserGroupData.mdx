# getUserGroupData

## Overview

The `getUserGroupData` API endpoint retrieves data about a specific user group. It is an HTTP GET endpoint that requires authentication.

## HTTP Method

- **Type**: `GET`

## Endpoint

- **URL**: `http://localhost:3000/v1/user/groups/data`
- **Description**: This endpoint returns data related to a user group specified by the `group_id`.
