## Overview

The `getUserGroupData` API endpoint allows you to retrieve data for a specific user group by providing the `group_id` as a query parameter. This endpoint requires bearer token authentication.

## Endpoint Details

- **Type**: HTTP GET
- **URL**: `http://localhost:3000/v1/user/groups/data`
- **Authentication**: Bearer Token

## Query Parameters

- **group_id** (required): The ID of the group to retrieve data for.
  - **Example**: `2`
