## Overview

The `getUserGroupData` API is an HTTP GET request designed to retrieve data related to user groups.

## HTTP Request

- **Method**: GET
- **URL**: `http://localhost:3000/v1/user/groups/data`

### Query Parameters

- `group_id`: A required query parameter to specify the ID of the group for which data is being requested.
  - **Example**: `group_id=2`
