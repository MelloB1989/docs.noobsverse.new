# getUserDetails API Documentation

This document provides details on the `getUserDetails` API, which retrieves user details using an HTTP GET request. It requires bearer token authentication.

## API Type

- **Type**: HTTP
- **Sequence**: 1

## HTTP GET Request

### URL

```
http://localhost:3000/v1/user/get
```

### Method

- **GET**

### Request Body

- **None**

### Authentication

The `getUserDetails` API requires authentication using a bearer token. Include the token in the `Authorization` header in the following format:

```
Authorization: Bearer <token>
```
