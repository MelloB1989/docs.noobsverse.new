## getUserDetails

The `getUserDetails` API is used to retrieve the details of a user. This is an HTTP GET request that requires a bearer token for authentication.

### HTTP Request

**GET** `http://localhost:3000/v1/user/get`

### Request Body

None. This API does not require a request body.
