# Get User Details API

## Overview

The `getUserDetails` API is used to fetch details of a user. It operates over HTTP and requires Bearer Token authentication.

## HTTP Request

### URL

```text
GET http://localhost:3000/v1/user/get
```

No body content is required for this request.
