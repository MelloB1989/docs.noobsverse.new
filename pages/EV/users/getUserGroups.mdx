# getUserGroups API

The `getUserGroups` API is an HTTP-based API used to retrieve the groups that a user belongs to. Authorization is required to access this API.

## Request

- **Method**: GET
- **URL**: `http://localhost:3000/v1/user/groups`
- **Body**: None
- **Authentication**: Bearer token
