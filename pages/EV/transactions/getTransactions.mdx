## getTransactions API

### Overview

This API retrieves all transactions.

### HTTP Method: GET

#### URL
`http://localhost:3000/v1/transactions/getAll`

#### Body
No request body is required for this API.

### Authorization

This endpoint requires a Bearer token for authorization.

#### Example Bearer Token

```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

Ensure to replace the example token with a valid token when making your request.

### Example Request

```http
GET /v1/transactions/getAll HTTP/1.1
Host: localhost:3000
Authorization: Bearer <your_token_here>
```
