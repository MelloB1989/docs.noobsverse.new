# getTransactions API

## Overview
The `getTransactions` API is an HTTP endpoint that retrieves a list of transactions. It requires bearer token authentication to access.

## Endpoint Details

### HTTP Method
**GET**

### URL
```
http://localhost:3000/v1/transactions
```

### Request Body
None

### Authentication
