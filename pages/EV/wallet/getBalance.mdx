## Overview

The `getBalance` API is a part of the HTTP-based services that allow users to retrieve their wallet balance information.

## Sequence

- **Sequence Number**: 2

## Endpoint Information

- **Type**: HTTP
- **Method**: GET
- **URL**: `http://localhost:3000/v1/wallet/get`
- **Request Body**: None

## Authentication

This API requires Bearer Token authentication.

### Bearer Token

To authenticate requests, you'll need to include a bearer token in the request header. The token format is as follows:

- **Authorization Header**: `Bearer <token>`

## Conclusion

The `getBalance` API enables the retrieval of wallet balance by using a simple HTTP GET request, secured via bearer token authentication. Ensure the token is valid and included in each request to successfully access the balance information.
