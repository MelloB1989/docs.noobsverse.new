## Overview

The **Get Balance** API is part of the wallet service suite. It provides functionality to retrieve the current balance of a specific wallet.

- **API Name:** getBalance
- **Type:** HTTP
- **Sequence:** 2

## Endpoint Details

### HTTP GET Request

- **URL:** `http://localhost:3000/v1/wallet/get`
- **Request Body:** None
- **Authentication:** Bearer Token
