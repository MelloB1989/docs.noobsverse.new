## getBalance API

### Overview

The `getBalance` API is a simple HTTP-based endpoint that allows users to fetch their wallet balance. This endpoint requires bearer authentication for access.

### Details

- **Name**: `getBalance`
- **Type**: HTTP
- **Sequence**: 2

### HTTP GET Request

#### URL

`http://localhost:3000/v1/wallet/get`

#### Body

No body is required for this request.
