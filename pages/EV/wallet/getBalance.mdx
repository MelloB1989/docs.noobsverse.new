# getBalance API

The `getBalance` API allows you to retrieve the balance of a user's wallet.

## HTTP GET Request

**URL:** `http://localhost:3000/v1/wallet/get`

**Method:** GET

**Body:** None

**Authentication:** Bearer Token

## Authentication

To use this API, you need to include a bearer token in the authorization header of your HTTP request. Below is an example token:
