## API Endpoint: Top Up Wallet

This API endpoint allows the user to top up their wallet balance.

### HTTP Method

**POST**

### URL

`http://localhost:3000/v1/wallet/topup`

### Authentication

This endpoint requires Bearer Token authentication.

### Request Body

The request body should be in JSON format.

#### JSON Body Example

```json
{
  "balance": "20"
}
```

### Sequence

This is the 3rd API call in the wallet service sequence.
