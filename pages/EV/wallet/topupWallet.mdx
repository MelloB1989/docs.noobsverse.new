## Overview

The `topupWallet` API endpoint allows you to top up a wallet by increasing its balance. This endpoint requires authentication through a Bearer token.

## Endpoint

- **URL:** `http://localhost:3000/v1/wallet/topup`
- **Method:** POST
- **Authentication:** Bearer Token

## Request Body

The request body must be in JSON format and should contain the following field:

- `balance`: A string representing the amount to top up the wallet balance.

### Example Request Body

```json
{
  "balance": "20"
}
```
