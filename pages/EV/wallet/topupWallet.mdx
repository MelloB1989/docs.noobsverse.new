# Topup Wallet API

## Overview

The Topup Wallet API allows you to add funds to a user's wallet. This endpoint requires authentication through a bearer token.

## Endpoint

### POST /v1/wallet/topup

- **URL**: `http://localhost:3000/v1/wallet/topup`
- **Method**: `POST`
- **Body**: JSON
- **Authentication**: Bearer token

## Authentication

To access this endpoint, you must include a bearer token in your request headers. Here's an example of a bearer token:

## Request Body

The request body should be in JSON format and include the following field:

- **balance**: The amount to add to the wallet as a string.

### Example

```json
{
  "balance": "20"
}
```

Make sure to replace the placeholder values with actual data as needed.
