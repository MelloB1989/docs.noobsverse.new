# API Documentation: Topup Wallet

## Overview

The `topupWallet` API allows users to add funds to their wallet. This operation is executed via an HTTP POST request.

- **Type**: HTTP
- **Sequence**: 3

## Endpoint

- **URL**: `http://localhost:3000/v1/wallet/topup`
- **Method**: POST

## Request Body

The body of the request should be in JSON format. It must contain the following field:

- `balance`: The amount to be added to the wallet. It is a string representing the monetary value.

### Example

```json
{
  "balance": "20"
}
```

## Response

On successful top up, the server will respond with the updated balance and a success message. The specific response structure is not detailed here; refer to API response documentation for more details.

Make sure to handle possible errors, such as authentication failure or invalid balance input, based on your application's requirements and server responses.
