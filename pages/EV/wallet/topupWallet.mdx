# Topup Wallet API

This API endpoint allows you to top up a user's wallet balance.

## Endpoint

### POST

- **URL**: `http://localhost:3000/v1/wallet/topup`
- **Body Format**: JSON
- **Authorization**: Bearer token

## Authentication

To access this endpoint, you must provide a valid bearer token in the authorization header.

```plaintext
Authorization: Bearer <token>
```

```plaintext

```

## Request Body

The request body must be in JSON format and should include the following field:

- **balance**: The amount to top up in the wallet.

### Example JSON Body

```json
{
  "balance": "20"
}
```

Ensure the `balance` is provided as a string representing the amount to add to the wallet balance.
