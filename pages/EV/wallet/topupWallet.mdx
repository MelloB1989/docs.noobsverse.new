`POST http://localhost:3000/v1/wallet/topup`

## Request Body
The request body should be in JSON format and must include the balance to top up:

```json
{
  "balance": "20"
}
```

## Example Request

## Response
On success, the server will return a status code of 200 OK along with the updated wallet details.

```json
{
  "message": "Wallet topped up successfully",
  "newBalance": 100
}
```

If the request fails, the server will return an appropriate error code with a message describing the error.
