# Top-up Wallet API

The `topupWallet` API allows you to top-up a user's wallet balance. This is an HTTP endpoint that requires a POST request with a JSON body and bearer authentication.

## Endpoint

```http
POST http://localhost:3000/v1/wallet/topup
```


## Request Body

The request should contain a JSON object with the following structure:

```json
{
  "balance": "20"
}
```

- `balance` (string): The amount to top-up the wallet.
