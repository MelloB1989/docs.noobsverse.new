# Topup Wallet API

This document provides detailed information about the Topup Wallet API.

## Endpoint Description

- **Name:** topupWallet
- **Type:** HTTP
- **Sequence:** 3

## HTTP Request

- **Method:** POST
- **URL:** `http://localhost:3000/v1/wallet/topup`
- **Body Format:** JSON

## Authentication

This API requires Bearer Token authentication. Include the following token in your request headers:

- **Authorization:** `Bearer <token>`

### Example Bearer Token

```plaintext
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiIwXzZpaGFjanJyIn0.Rf1ZUJpJdBBbep4cGCqwBQ2A8IyETp2UTrnecGHdi6I
```

## Request Body

The request body should be in JSON format and include the following field:

- **balance:** (string) The amount to top up the wallet.

### Example Request Body

```json
{
  "balance": "20"
}
```

Make sure to provide a valid JSON object with the `balance` field when making the POST request to the above URL.
