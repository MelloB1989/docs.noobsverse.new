# topupWallet API Documentation

## Overview

The `topupWallet` API allows users to top up their wallet balance. This is an HTTP POST request that requires bearer token authentication and a JSON body specifying the top-up amount.

## Endpoint

**URL:** `http://localhost:3000/v1/wallet/topup`

**Method:** POST

## Request Body

The request body should be in JSON format and must include the following parameter:

| Parameter  | Type   | Description           |
|------------|--------|-----------------------|
| `balance`  | string | The amount to top up. |

**Example:**

```json
{
  "balance": "20"
}
```
