# topupWallet

This endpoint allows you to top up a wallet with a specified balance.

## Endpoint

**URL:** `http://localhost:3000/v1/wallet/topup`

**Method:** POST

**Authentication:** Bearer Token

## Request Body

Send a JSON object in the body of the request with the structure below:

```json
{
  "balance": "20"
}
```

- `balance`: The amount to top up the wallet.
