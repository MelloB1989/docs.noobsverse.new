# Webhook API

The Webhook API allows you to send events or updates to a specified URL. Below are the details for using this endpoint.

## HTTP GET Request

### URL

`http://localhost:3000/v1/wallet/webhook`

### Query Parameters

| Name           | Type   | Description                     | Example                                          |
| -------------- | ------ | ------------------------------- | ------------------------------------------------ |
| `uid`          | string | Unique user identifier          | `5geapgapxs`                                     |
| `balance`      | string | Balance value                   | `10000000000`                                    |
| `webhook_key`  | string | Key for webhook authentication  | `coffeecodesEVBackendKarmafdmfbmd3brh4rfembvfadsadmbdbbvda` |

### Authentication

This endpoint requires bearer token authentication. Include the token in the `Authorization` header:

```
Authorization: Bearer <TOKEN>
```

Example token format:
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiIwXzZpaGFjanJyIn0.Rf1ZUJpJdBBbep4cGCqwBQ2A8IyETp2UTrnecGHdi6I
```
