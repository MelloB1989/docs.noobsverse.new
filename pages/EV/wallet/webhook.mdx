# Webhook API

The Webhook API allows you to receive updates about events related to wallets.

## Endpoint

### GET
- **URL:** `http://localhost:3000/v1/wallet/webhook`
- **Authorization:** Bearer Token

## Query Parameters

| Parameter      | Type   | Description                                    | Example                                      |
| -------------- | ------ | ---------------------------------------------- | -------------------------------------------- |
| `uid`          | string | Unique identifier for the user                 | `5geapgapxs`                                 |
| `balance`      | string | Current balance of the wallet                  | `10000000000`                                |
| `webhook_key`  | string | Key used to identify the webhook               | `coffeecodesEVBackendKarmafdmfbmd3brh4rfembvfadsadmbdbbvda` |
