# Webhook API

This API allows you to trigger a webhook with specific parameters. It is an HTTP-based API which requires Bearer authentication.

## Endpoint

### GET Request
```
GET http://localhost:3000/v1/wallet/webhook
```

### Query Parameters

| Name         | Type   | Description                              |
| ------------ | ------ | ---------------------------------------- |
| `uid`        | string | Unique identifier for the user           |
| `balance`    | number | The balance amount                       |
| `webhook_key`| string | Key specifically used for webhook access |

**Example URL:**
```
http://localhost:3000/v1/wallet/webhook?uid=5geapgapxs&balance=10000000000&webhook_key=coffeecodesEVBackendKarmafdmfbmd3brh4rfembvfadsadmbdbbvda
```
