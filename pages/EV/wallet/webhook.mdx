# Webhook API

This document provides details for the Webhook API endpoint.

## Endpoint

### HTTP Method

**GET**

### URL

```
http://localhost:3000/v1/wallet/webhook
```

## Query Parameters

The following query parameters are used by this endpoint:

- **uid**: User identifier. Example: `5geapgapxs`
- **balance**: Balance amount. Example: `10000000000`
- **webhook_key**: Key for webhook authentication. Example: `coffeecodesEVBackendKarmafdmfbmd3brh4rfembvfadsadmbdbbvda`

### Full Example

```
http://localhost:3000/v1/wallet/webhook?uid=5geapgapxs&balance=10000000000&webhook_key=coffeecodesEVBackendKarmafdmfbmd3brh4rfembvfadsadmbdbbvda
```

## Request Body

This request does not require a body.

## Summary

The Webhook API is a GET endpoint that requires query parameters and bearer token authentication. It allows interaction with a user's wallet by providing information including user ID (`uid`), account balance (`balance`), and a specific webhook key (`webhook_key`).
