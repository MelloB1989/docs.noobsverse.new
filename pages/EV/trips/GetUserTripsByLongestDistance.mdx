# GetUserTripsByLongestDistance

This API endpoint fetches user trips sorted by the longest distance.

## HTTP Request

### GET

`http://localhost:3000/v1/trips/distance/longest?limit=10`

### Query Parameters

- `limit`: (optional) The maximum number of trips to return. Default is 10.

### Authentication

This endpoint requires Bearer Token authentication.

## Response
