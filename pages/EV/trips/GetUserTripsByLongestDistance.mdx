## Overview

The `GetUserTripsByLongestDistance` API endpoint retrieves a list of user trips sorted by the longest distance. This endpoint is useful for fetching trips based on distance, providing insights into user travel patterns.

## HTTP Method and URL

- **Method**: `GET`
- **URL**: `http://localhost:3000/v1/trips/distance/longest?limit=10`

Example: `?limit=10`
