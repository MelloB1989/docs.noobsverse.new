# GetUserTripsByLongestDistance API

The `GetUserTripsByLongestDistance` API retrieves user trips sorted by the longest distance. This endpoint allows you to specify the number of trips to return.

## HTTP Request

```http
GET http://localhost:3000/v1/trips/distance/longest?limit=10
```

## Query Parameters

- `limit` (integer) - The number of trips to retrieve. Default is 10.

## Request Headers

- `Authorization` - Bearer token for authentication. Example:
  ```

  ```
