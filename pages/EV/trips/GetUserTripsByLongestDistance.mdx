# GetUserTripsByLongestDistance

## Overview
The `GetUserTripsByLongestDistance` API endpoint retrieves a list of user trips sorted by the longest distance. The response is limited to a specified number of trips.

## HTTP Request

### GET
```http
GET http://localhost:3000/v1/trips/distance/longest?limit=10
```

## Query Parameters

| Parameter | Type   | Description                             |
|-----------|--------|-----------------------------------------|
| limit     | Number | (Optional) Limits the number of results |

- **Default value:** 10
