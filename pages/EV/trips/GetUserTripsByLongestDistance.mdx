# GetUserTripsByLongestDistance

## Description

Retrieve user trips sorted by the longest distance. 

## Endpoint

**GET** `http://localhost:3000/v1/trips/distance/longest?limit=10`

## Query Parameters

| Name  | Type | Default | Description                     |
|-------|------|---------|---------------------------------|
| limit | int  | 10      | The number of trips to retrieve |
