## GetUserTripsByLongestDistance API

### Description

This API retrieves user trips sorted by the longest distance. It allows you to set a limit on the number of trips returned.

### HTTP Request

`GET http://localhost:3000/v1/trips/distance/longest`

### Query Parameters

- `limit` (integer): The maximum number of trips to return.
  - **Default:** 10
  - **Required:** No
