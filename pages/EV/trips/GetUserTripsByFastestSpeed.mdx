## Endpoint Overview

### Name
GetUserTripsByFastestSpeed

### Type
HTTP

### Sequence
10

## HTTP Method
`GET`

## URL
```
http://localhost:3000/v1/trips/speed/fastest?limit=10
```

## Description
This endpoint retrieves a list of user trips sorted by the fastest speed, limited to a specified number of records.

## Request Details

### Query Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| limit     | int  | true     | The maximum number of trips to retrieve. Default is 10. |

Example:
```
?limit=10
```
