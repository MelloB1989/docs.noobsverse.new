## GetUserTripsByFastestSpeed

Retrieve the user's trips sorted by the fastest speed.

### HTTP Request

`GET http://localhost:3000/v1/trips/speed/fastest?limit=10`

### Query Parameters

| Parameter | Type | Description          | Required | Default |
|-----------|------|----------------------|----------|---------|
| limit     | int  | Limit the number of results | Yes      | 10      |
