## Overview

The **GetUserTripsByFastestSpeed** API endpoint retrieves user trips sorted by the fastest speed. This endpoint is part of the `/v1/trips/speed` namespace and accesses the `fastest` resource.

## HTTP Request

**GET** `http://localhost:3000/v1/trips/speed/fastest?limit=10`

## Query Parameters

| Name  | Type | Required | Description                          |
|-------|------|----------|--------------------------------------|
| limit | int  | Yes      | The number of trips to retrieve. Default is 10. |

## Error Handling

- **401 Unauthorized**: Returned if the bearer token is missing or invalid.
- **400 Bad Request**: Returned if the query parameter `limit` is invalid.

## Notes

- Ensure the bearer token is valid and up to date to avoid authentication errors.
