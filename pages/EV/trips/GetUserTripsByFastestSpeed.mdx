# GetUserTripsByFastestSpeed API

## Description

Fetches the user's trips ordered by the fastest speed. The result can be limited by a query parameter.

## HTTP Request

**GET** `http://localhost:3000/v1/trips/speed/fastest?limit=10`

## Parameters

### Query Parameters

- `limit` (optional): The maximum number of trips to return. Default is set to 10.

This endpoint requires Bearer authentication.
