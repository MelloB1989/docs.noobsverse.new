# GetUserTripsByFastestSpeed API

This endpoint retrieves a list of user trips sorted by the fastest speed achieved. The data is limited to a specified number of trips.

## HTTP Request

### GET

`GET http://localhost:3000/v1/trips/speed/fastest?limit=10`

- **URL**: `http://localhost:3000/v1/trips/speed/fastest`
- **Method**: `GET`
- **Query Parameter**: `limit`
  - **Type**: Integer
  - **Default**: 10
