# GetUserTripsByFastestSpeed API Documentation

## Overview

The **GetUserTripsByFastestSpeed** endpoint retrieves a list of user trips sorted by the fastest speed. The results can be limited using query parameters.

## HTTP Method and URL

- **GET** : `http://localhost:3000/v1/trips/speed/fastest?limit=10`

- **400 Bad Request**: Returned when the request parameters are incorrect.
