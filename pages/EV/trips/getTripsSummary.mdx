# `getTripsSummary` API

The `getTripsSummary` API is an HTTP-based API used to retrieve a summary of trips for the authenticated user.

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/mytrips`
- **Authentication:** Bearer Token
- **Request Body:** None
