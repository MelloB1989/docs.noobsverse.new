# Start Trip API

Initiate a trip using this API. This endpoint allows the user to start a trip associated with a bike and a station.

## Endpoint

**POST** `http://localhost:3000/v1/trips/start`

## Authentication

This endpoint requires Bearer token authentication.


## Request Body

Send the following JSON payload in the body of the request:

```json
{
    "bike_id": "i6bjwk0yhu",
    "station_id": "a2m392c58f",
    "start_timestamp": "2024-09-02T07:15:00Z"
}
```

### Parameters

- **bike_id** (string): ID of the bike starting the trip.
- **station_id** (string): ID of the station from which the trip starts.
- **start_timestamp** (string): ISO 8601 formatted timestamp indicating the start time of the trip. 

Ensure the request header includes `Content-Type: application/json` and the Authorization header is set to `Bearer <your-token>`.
