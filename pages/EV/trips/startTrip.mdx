# Start Trip API

## Endpoint

**POST** `http://localhost:3000/v1/trips/start`

## Description

Initiates a trip for a bike from a specified station at a given start time.

This endpoint requires Bearer Token authentication.


## Request Body

The request body must be sent in JSON format and include the following fields:

```json
{
    "bike_id": "i6bjwk0yhu",
    "station_id": "a2m392c58f",
    "start_timestamp": "2024-09-02T07:15:00Z"
}
```

- **bike_id**: *string* (required) - The unique identifier of the bike.
- **station_id**: *string* (required) - The unique identifier of the station.
- **start_timestamp**: *string* (required) - The timestamp when the trip starts, in ISO 8601 format.
