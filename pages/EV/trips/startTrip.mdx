# Start Trip API

The `startTrip` API is an HTTP endpoint for initiating a trip in the system. Below are the details required to successfully call this API.

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/start`
- **Method:** `POST`

## Authorization

This endpoint requires Bearer Token authorization.


## Request Body

The body of the request should be in JSON format. Below is the required structure:

```json
{
  "bike_id": "i6bjwk0yhu",
  "station_id": "a2m392c58f",
  "start_timestamp": "2024-09-02T07:15:00Z"
}
```

- **bike_id**: The identifier of the bike being used for the trip.
- **station_id**: The identifier of the station where the trip starts.
- **start_timestamp**: The timestamp indicating when the trip starts, in ISO 8601 format.

Ensure that all required fields are included and correctly formatted in your request to successfully start a trip.
