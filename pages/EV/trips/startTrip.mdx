# startTrip API

## Description

This endpoint is used to start a trip with a bike from a specific station. The request must be authenticated using a bearer token.

## HTTP Request

**POST** `http://localhost:3000/v1/trips/start`

## Authentication

This request requires a bearer token for authentication.

## Request Body

The body of the request should be in JSON format.

```json
{
    "bike_id": "i6bjwk0yhu",
    "station_id": "a2m392c58f",
    "start_timestamp": "2024-09-02T07:15:00Z"
}
```

### Parameters

- **bike_id**: The identifier of the bike (string).
- **station_id**: The identifier of the station where the trip starts (string).
- **start_timestamp**: The timestamp when the trip starts in ISO 8601 format (string).

## Response

A successful response will start the trip and return the details of the trip started. The exact response structure should be specified in your API documentation.
