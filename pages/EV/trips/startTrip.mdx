## startTrip API

The `startTrip` API initiates a new bike trip by specifying the bike ID, the station ID where the trip starts, and the start timestamp.

### Endpoint

**POST** `http://localhost:3000/v1/trips/start`

This endpoint uses Bearer token authentication. Include the following token in the Authorization header of your request:


### Request Body

The request body should be in JSON format and include the following fields:

- **bike_id**: The unique identifier for the bike.
- **station_id**: The unique identifier for the station where the trip starts.
- **start_timestamp**: The ISO 8601 formatted timestamp indicating when the trip starts.

Example request body:

```json
{
  "bike_id": "i6bjwk0yhu",
  "station_id": "a2m392c58f",
  "start_timestamp": "2024-09-02T07:15:00Z"
}
```
