## startTrip

### Overview

The `startTrip` API allows you to start a new trip for a specific bike from a specified station at a given start time.

## Endpoint

### HTTP Request

`POST http://localhost:3000/v1/trips/start`

### Request Headers

- **Authorization**: Bearer Token

### Bearer Token


### Request Body

The request body must be in JSON format and include the following fields:

```json
{
  "bike_id": "i6bjwk0yhu",
  "station_id": "a2m392c58f",
  "start_timestamp": "2024-09-02T07:15:00Z"
}
```

- **bike_id**: (string) The ID of the bike.
- **station_id**: (string) The ID of the station from which the trip is starting.
- **start_timestamp**: (string) The start time of the trip in ISO 8601 format.
