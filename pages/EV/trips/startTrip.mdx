# Start Trip API

This API allows users to start a trip by posting the necessary trip details.

## Endpoint

**POST** `http://localhost:3000/v1/trips/start`

This API uses Bearer Token authentication. Include the following token in the `Authorization` header:


## Request Body

The request body should be in JSON format and include the following fields:

```json
{
  "bike_id": "i6bjwk0yhu",
  "station_id": "a2m392c58f",
  "start_timestamp": "2024-09-02T07:15:00Z"
}
```

- `bike_id` (string) - The unique identifier of the bike.
- `station_id` (string) - The unique identifier of the station.
- `start_timestamp` (string) - The timestamp when the trip starts in ISO 8601 format.
