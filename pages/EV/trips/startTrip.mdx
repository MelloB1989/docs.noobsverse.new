# Start Trip API

This endpoint is used to initiate a bike trip. It requires authentication and specific JSON data in the request body.

## HTTP Request

### POST
`http://localhost:3000/v1/trips/start`

### Authentication

This endpoint requires a bearer token for authentication.

### Request Body

The request body must be in JSON format and include the following fields:

| Field           | Type   | Description                                |
|-----------------|--------|--------------------------------------------|
| `bike_id`       | string | The unique identifier for the bike.        |
| `station_id`    | string | The unique identifier for the station.     |
| `start_timestamp` | string | The timestamp marking the trip's start time in ISO 8601 format. |

#### Example JSON Body:
```json
{
    "bike_id": "i6bjwk0yhu",
    "station_id": "a2m392c58f",
    "start_timestamp": "2024-09-02T07:15:00Z"
}
```
