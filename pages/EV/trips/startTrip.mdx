# Start Trip API

This document outlines the details for the `startTrip` API endpoint, which is responsible for beginning a new bike trip.

## HTTP Method

**POST** 

## Endpoint

**URL:** `http://localhost:3000/v1/trips/start`

## Request Body

The body of the request should be in JSON format with the following structure:

```json
{
  "bike_id": "i6bjwk0yhu",
  "station_id": "a2m392c58f",
  "start_timestamp": "2024-09-02T07:15:00Z"
}
```

### Parameters

- `bike_id`: The unique identifier for the bike being rented. *(String)*
- `station_id`: The unique identifier for the station where the trip is starting. *(String)*
- `start_timestamp`: The ISO 8601 formatted date-time string representing when the trip begins. *(String)*
