# GetUserTripsByLongestDuration

## Endpoint

- **Type**: `GET`
- **URL**: `http://localhost:3000/v1/trips/duration/longest?limit=10`
- **Authentication**: Bearer Token

## Parameters

### Query Parameters

- **limit**: 

  - **Type**: integer
  - **Default**: 10
  - **Description**: The maximum number of trips to return, sorted by longest duration.
