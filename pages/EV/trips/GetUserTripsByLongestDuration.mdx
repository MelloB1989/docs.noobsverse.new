# GetUserTripsByLongestDuration

## Overview

The `GetUserTripsByLongestDuration` API endpoint retrieves a list of user trips sorted by the longest duration. The endpoint provides the ability to limit the number of trips returned in the response.

## HTTP Request

- **Type:** GET
- **URL:** `http://localhost:3000/v1/trips/duration/longest?limit=10`
