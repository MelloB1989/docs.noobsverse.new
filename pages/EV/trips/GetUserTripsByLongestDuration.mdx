# GetUserTripsByLongestDuration API

This API endpoint retrieves the user's trips sorted by the longest duration.

## Endpoint Overview

- **Name**: GetUserTripsByLongestDuration
- **Type**: HTTP
- **Sequence**: 8

## HTTP Request

### GET

- **URL**: `http://localhost:3000/v1/trips/duration/longest?limit=10`
- **Body**: None
- **Authentication**: Bearer Token

### Query Parameters

| Name  | Type  | Required | Description                              |
|-------|-------|----------|------------------------------------------|
| limit | Query | Yes      | Specifies the number of trips to return. |

#### Example Query

- `limit=10`
