## GetUserTripsByLongestDuration

### Description

This endpoint retrieves user trips, sorted by the longest duration. It allows you to limit the number of trips retrieved.

### HTTP Request

- **Method**: `GET`
- **URL**: `http://localhost:3000/v1/trips/duration/longest?limit=10`

### Request Parameters

#### Query Parameters

| Parameter | Type | Description                             | Example |
|-----------|------|-----------------------------------------|---------|
| limit     | int  | The maximum number of trips to retrieve | 10      |
