# GetUserTripsByLongestDuration

The `GetUserTripsByLongestDuration` API retrieves the user's trips sorted by the longest duration.

## HTTP Request

### GET

```http
GET http://localhost:3000/v1/trips/duration/longest?limit=10
```

### Query Parameters

| Name  | Type | Description       | Default |
|-------|------|-------------------|---------|
| limit | int  | Maximum number of trips to retrieve | 10      |

## Response

The response will contain an array of trips sorted by the longest duration.

**Status Code:** 200 OK

**Response Body:**

```json
[
  {
    "tripId": "string",
    "duration": "number",
    "otherTripDetails": "object"
  },
  // More trip objects...
]
```

Make sure to replace the bearer token with an appropriate token when making requests to this endpoint.
