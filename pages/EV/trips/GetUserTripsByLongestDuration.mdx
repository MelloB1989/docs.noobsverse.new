# GetUserTripsByLongestDuration API

## Overview

This API endpoint retrieves the user trips sorted by the longest duration.

## HTTP Method

- **GET** 

## URL

- `http://localhost:3000/v1/trips/duration/longest`

## Query Parameters

| Name  | Type | Required | Description                           |
|-------|------|----------|---------------------------------------|
| limit | int  | Optional | Limits the number of records returned |

### Example

- **URL with Query Parameter**:
  ```
  http://localhost:3000/v1/trips/duration/longest?limit=10
  ```

## Request Body

- No request body needed.
