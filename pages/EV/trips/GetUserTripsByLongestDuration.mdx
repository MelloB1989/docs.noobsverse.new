## Overview

`GetUserTripsByLongestDuration` is an HTTP GET endpoint that retrieves user trips sorted by the longest duration. The endpoint allows specifying a limit to control the number of trips returned.

## Endpoint

### URL

`http://localhost:3000/v1/trips/duration/longest`

### Method

`GET`

## Query Parameters

| Name  | Type | Description                     | Default |
|-------|------|---------------------------------|---------|
| limit | int  | The number of trips to retrieve | `10`    |

### Example

```plaintext
http://localhost:3000/v1/trips/duration/longest?limit=10
```
