# GetUserTripsByLongestDuration API

## Endpoint

**GET** `http://localhost:3000/v1/trips/duration/longest`

This endpoint retrieves user trips sorted by the longest duration, with an optional limit on the number of results.

## Parameters

### Query Parameters

- `limit` (integer, optional): The maximum number of results to return. Default value is `10`.
