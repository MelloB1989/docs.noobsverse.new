## Request Body

The following JSON object should be included in the request body:

```json

    "id": "ulj1iq0-0w",
    "bike_id": "i6bjwk0yhu",
    "station_id": "a2m392c58f",
    "start_timestamp": "2024-09-02T07:15:00Z",
    "end_timestamp": "2024-09-02T07:50:00Z",

    "duration": 35.0,
    "average_speed": 27.1,
    "path": "base64_encoded_path_data_3"

```

### Fields Description

- **id**: Unique identifier for the trip (string).
- **bike_id**: Unique identifier for the bike used in the trip (string).
- **station_id**: Unique identifier for the station where the trip started and ended (string).
- **start_timestamp**: ISO 8601 formatted timestamp indicating when the trip started (string).
- **end_timestamp**: ISO 8601 formatted timestamp indicating when the trip ended (string).
- **distance**: Total distance covered during the trip, in kilometers (number).
- **duration**: Total duration of the trip, in minutes (number).
- **average_speed**: Average speed during the trip, in km/h (number).
- **path**: Base64 encoded data representing the path taken during the trip (string).
