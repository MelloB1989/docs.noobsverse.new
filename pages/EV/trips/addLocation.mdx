# Add Location API

`addLocation` is an HTTP API that allows users to add a new location to a specific trip.

## Endpoint

**POST** `http://localhost:3000/v1/trips/ulj1iq0-0w/location`

## Authentication

This API requires Bearer Token authentication.

### Bearer Token

```plaintext
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

Include this token in the Authorization header of your HTTP request:

```
Authorization: Bearer <token>
```

## Request Body

The request body must be in JSON format, containing the latitude and longitude of the location:

```json
{
  "lat": 37.776,
  "long": -122.426
}
```

- `lat` (float): The latitude of the location.
- `long` (float): The longitude of the location.


Use this API to add locations to your trips, ensuring you include the required authentication and correct parameters in your request body.
