# Add Location API

The `addLocation` endpoint allows you to add a new location to an existing trip.

## Endpoint Details

- **Type:** HTTP
- **Sequence:** 13
- **Method:** POST

### URL

`http://localhost:3000/v1/trips/ulj1iq0-0w/location`


Ensure to include the token in the `Authorization` header as follows:

```
Authorization: Bearer <token>
```

## Request Body

The request body should be in JSON format and must include the following fields:

```json
{
  "lat": 37.776,
  "long": -122.426
}
```

- **lat**: Latitude of the location.
- **long**: Longitude of the location.
