# Add Location

Adds a location to an existing trip.

## Endpoint

**POST** `http://localhost:3000/v1/trips/ulj1iq0-0w/location`

## Authentication

This endpoint requires a Bearer Token for authentication.

## Request Body

The request body should be in JSON format and contain the following fields:

```json
{
  "lat": 37.776,
  "long": -122.426
}
```

- **lat**: Latitude of the location (e.g., `37.776`).
- **long**: Longitude of the location (e.g., `-122.426`).
