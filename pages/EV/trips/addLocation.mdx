# addLocation API

## Overview
The `addLocation` API is an HTTP endpoint used to add a location to a specific trip. It is designed to handle `POST` requests and requires authentication via a bearer token.

## Endpoint
- **URL**: `http://localhost:3000/v1/trips/ulj1iq0-0w/location`
- **Method**: `POST`
- **Authentication**: Bearer Token

## Authentication

## Request Body
The request must include a JSON payload with the following structure:

```json
{
  "lat": 37.776,
  "long": -122.426
}
```

- **lat**: Latitude of the location (e.g., `37.776`).
- **long**: Longitude of the location (e.g., `-122.426`).

## Response
The API will provide a response indicating the success or failure of the location addition. The response format and status codes will depend on the implementation details not provided here.
