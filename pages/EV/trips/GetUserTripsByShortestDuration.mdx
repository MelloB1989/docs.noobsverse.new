# GetUserTripsByShortestDuration

## Endpoint

GET `http://localhost:3000/v1/trips/duration/shortest`

## Description

Retrieve user trips ordered by the shortest duration.

## Parameters

### Query Parameters

- `limit`: Maximum number of trips to retrieve. Default is `10`.

```http
GET /v1/trips/duration/shortest?limit=10
```
