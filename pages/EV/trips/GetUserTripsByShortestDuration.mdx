# GetUserTripsByShortestDuration

Retrieves a list of user trips ordered by the shortest duration.

## HTTP Request

### GET

`http://localhost:3000/v1/trips/duration/shortest?limit=10`

## Query Parameters

- `limit` (optional, default: 10): The maximum number of trips to return.
