# GetUserTripsByShortestDuration API

## Description

This API endpoint retrieves user trips sorted by the shortest duration. It allows you to limit the number of results returned.

## Endpoint

### GET

`http://localhost:3000/v1/trips/duration/shortest?limit=10`

## Query Parameters

| Name  | Type | Required | Description                          |
|-------|------|----------|--------------------------------------|
| limit | int  | No       | The maximum number of trips to return (default: 10) |
