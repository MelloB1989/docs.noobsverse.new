# GetUserTripsByShortestDuration API

The `GetUserTripsByShortestDuration` API provides the capability to retrieve a list of user trips sorted by the shortest duration.

## Sequence

- API Sequence: 9

## HTTP Method

- **GET**

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/duration/shortest?limit=10`

## Request Details

- **Request Body:** None
