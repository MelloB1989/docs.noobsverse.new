# GetUserTripsByShortestDuration API

## Overview

The `GetUserTripsByShortestDuration` API endpoint retrieves user trips ordered by the shortest duration. This document provides the necessary information for interacting with this endpoint.

## HTTP Request

### GET Request

- **URL:** `http://localhost:3000/v1/trips/duration/shortest?limit=10`
- **Method:** GET
- **Authentication:** Bearer Token

```http
GET http://localhost:3000/v1/trips/duration/shortest?limit=10
Authorization: Bearer <token>
```

### Query Parameters

- `limit`: Specifies the maximum number of trips to return (default: 10).
