# GetUserTripsByShortestDuration API

This API retrieves user trips sorted by the shortest duration.

## HTTP Request

**GET** `http://localhost:3000/v1/trips/duration/shortest?limit=10`

## Authentication

This endpoint requires a Bearer token for authentication.

**Header:**

```plaintext
Authorization: Bearer {token}
```

**Example Token:**

```plaintext
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

## Query Parameters

| Parameter | Type  | Description                                   | Required | Default Value |
|-----------|-------|-----------------------------------------------|----------|---------------|
| limit     | number| The number of trips to retrieve (maximum: 10) | yes      | 10            |

## Example Request

```plaintext
GET http://localhost:3000/v1/trips/duration/shortest?limit=10
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

> Ensure that you replace the `{token}` in the Authorization header with your actual Bearer token.

## Response

The response will contain the trips sorted by the shortest duration based on the limit specified.
