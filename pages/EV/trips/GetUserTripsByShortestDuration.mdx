# GetUserTripsByShortestDuration

## Overview

The `GetUserTripsByShortestDuration` API endpoint retrieves a list of user trips sorted by the shortest duration. This documentation provides details on how to use this API endpoint.

## Endpoint Details

- **Type**: HTTP
- **Method**: `GET`
- **URL**: `http://localhost:3000/v1/trips/duration/shortest?limit=10`
- **Authentication**: Bearer Token

## Response

The response will include a JSON object with a list of trips sorted by the shortest duration.

## Status Codes

- **200 OK**: The request was successful, and the response body contains the list of trips.
- **401 Unauthorized**: The request lacks valid authentication credentials.
- **400 Bad Request**: The request parameters are invalid.
- **500 Internal Server Error**: An error occurred on the server.

Please ensure that your request includes the necessary bearer token and query parameters to successfully retrieve the trips.
