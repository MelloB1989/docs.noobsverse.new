# GetUserTripsByShortestDuration

This API retrieves user trips sorted by shortest duration.

## HTTP Request

### GET
**URL:** `http://localhost:3000/v1/trips/duration/shortest?limit=10`

**Body:** None

**Authentication:** Bearer Token

## Query Parameters

| Parameter | Type | Description               | Example |
|-----------|------|---------------------------|---------|
| limit     | int  | The maximum number of results to retrieve | `10`     |
