# getLocations API

The `getLocations` API is used to retrieve a list of locations for a specific trip.

## HTTP Method

**GET**

## Endpoint

```
http://localhost:3000/v1/trips/ulj1iq0-0w/locations
```

## Request Body

No request body is required.
