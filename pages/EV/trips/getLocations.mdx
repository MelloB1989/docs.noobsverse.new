# getLocations API

## Overview

The `getLocations` API allows users to retrieve the locations for a specific trip using an HTTP GET request. This endpoint is secured with bearer token authentication.

## HTTP Method

- **GET**

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/ulj1iq0-0w/locations`

## Response

The response will contain the locations associated with the specified trip. The structure of the response is determined by the implementation of the API.

## Common Status Codes

- **200 OK:** The request was successful, and the locations are returned.
- **401 Unauthorized:** The bearer token is missing or invalid.
- **404 Not Found:** The specified trip or locations were not found.
