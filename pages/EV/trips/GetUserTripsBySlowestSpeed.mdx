## API Overview

`GetUserTripsBySlowestSpeed` is an HTTP GET API that retrieves a list of user trips sorted by the slowest speed. The endpoint allows you to specify the number of trips to retrieve via a query parameter.

---

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/speed/slowest?limit=10`
- **Method:** `GET`
- **Authentication:** Bearer Token
- **Request Body:** None

---

## Query Parameters

| Name  | Type | Description               | Required | Default |
|-------|------|---------------------------|----------|---------|
| limit | int  | Number of trips to return | Yes      | 10      |

- **`limit`**: Specifies the number of trips to be retrieved. The default value is `10`.
