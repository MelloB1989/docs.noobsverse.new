## GetUserTripsBySlowestSpeed API

### Description

This API retrieves a list of user trips sorted by the slowest speed. The results are limited by a specified number of trips.

### HTTP Request

`GET http://localhost:3000/v1/trips/speed/slowest?limit=10`

### Authentication

This endpoint requires a bearer token for authentication.

### Query Parameters

| Name  | Type | Required | Description                          |
|-------|------|----------|--------------------------------------|
| limit | int  | No       | Maximum number of trips to retrieve. |

* **Default Value**: `10`
