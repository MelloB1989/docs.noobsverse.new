# GetUserTripsBySlowestSpeed

This API endpoint retrieves a list of user trips sorted by the slowest speed.

## HTTP Method: GET

### URL 

`http://localhost:3000/v1/trips/speed/slowest?limit=10`

### Headers

- `Authorization: Bearer YOUR_JWT_TOKEN`

### Query Parameters

- `limit` (default: `10`): The maximum number of trip records to return.

This endpoint requires Bearer Token authentication.
