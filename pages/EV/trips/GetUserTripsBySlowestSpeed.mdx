## GetUserTripsBySlowestSpeed

### Description

Fetches user trips sorted by the slowest speed, limited to a specified number of results.

### HTTP Request

**GET** `http://localhost:3000/v1/trips/speed/slowest?limit=10`

### Authentication

This endpoint requires a Bearer token for authentication.


### Parameters

#### Query Parameters

| Name | Type | Description         | Required | Default |
|------|------|---------------------|----------|---------|
| limit | int  | Number of results to return. | yes      | 10      |

### Response

Returns a list of user trips sorted by the slowest speed.

### Example Request

```shell
curl -X GET "http://localhost:3000/v1/trips/speed/slowest?limit=10" -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw"
```
