## GetUserTripsBySlowestSpeed API

This API endpoint retrieves user trips sorted by the slowest speed, with a specified limit on the number of trips returned.

### HTTP Request

**GET** `http://localhost:3000/v1/trips/speed/slowest?limit=10`

### Parameters

| Name  | Type   | Location | Description                 |
|-------|--------|----------|-----------------------------|
| limit | number | query    | Maximum number of trips to return. Default is 10. |

### Request Body

This endpoint does not require a request body.
