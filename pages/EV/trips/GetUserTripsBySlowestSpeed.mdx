# GetUserTripsBySlowestSpeed

This API retrieves user trips sorted by the slowest speed.

## Endpoint

### HTTP GET

`http://localhost:3000/v1/trips/speed/slowest`

## Query Parameters

- `limit`: Optional integer query parameter to limit the number of results returned. Default value is `10`.
