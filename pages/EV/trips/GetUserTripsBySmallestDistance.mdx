# GetUserTripsBySmallestDistance API

This API endpoint retrieves user trips sorted by the shortest distance. It is designed to return a list of trips with the option to specify the number of records via query parameters.

## Endpoint Details

- **Type:** HTTP GET
- **URL:** `http://localhost:3000/v1/trips/distance/shortest`

## Query Parameters

- **`limit`:** *(optional)* Specifies the maximum number of trips to return. Default is 10.

Example: `?limit=10`
