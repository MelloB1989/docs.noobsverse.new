# GetUserTripsBySmallestDistance

This endpoint retrieves a list of user trips sorted by the shortest distance. The response is limited to a specified number of trips.

## HTTP Request

`GET http://localhost:3000/v1/trips/distance/shortest?limit=10`

## Parameters

### Query Parameters

| Name  | Type | Description                                      |
|-------|------|--------------------------------------------------|
| limit | int  | The maximum number of trips to return. (default: 10) |
