# GetUserTripsBySmallestDistance API

## Description

This API endpoint retrieves user trips sorted by the smallest distance. The endpoint accepts an optional limit parameter specifying the maximum number of trips to return.

## HTTP Method

### GET

**URL:**

`http://localhost:3000/v1/trips/distance/shortest?limit=10`

**Body:**

None

**Authentication:**

Bearer

## Parameters

### Query Parameters

- **limit**: (Optional) The maximum number of trips to return.
