## GetUserTripsBySmallestDistance

This API retrieves user trips ordered by the shortest distance.

### Endpoint

`GET http://localhost:3000/v1/trips/distance/shortest`

### Query Parameters

| Name | Type | Description                    | Required | Default |
|------|------|--------------------------------|----------|---------|
| limit| int  | The number of trips to retrieve| No       | 10      |

### Authentication

This endpoint requires a bearer token.
