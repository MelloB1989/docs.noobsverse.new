## GetUserTripsBySmallestDistance

`GetUserTripsBySmallestDistance` is an HTTP GET API used to retrieve a list of user trips sorted by the smallest distance.

### Endpoint

- **URL:** `http://localhost:3000/v1/trips/distance/shortest?limit=10`
- **Method:** `GET`

### Query Parameters

| Parameter | Type   | Description          | Required | Default |
|-----------|--------|----------------------|----------|---------|
| limit     | number | Number of trips to return | No       | 10      |
