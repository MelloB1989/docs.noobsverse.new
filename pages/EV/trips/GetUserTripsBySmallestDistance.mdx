# GetUserTripsBySmallestDistance API

This API retrieves a list of user trips ordered by the shortest distance.

## HTTP Method

**GET**

## URL

`http://localhost:3000/v1/trips/distance/shortest?limit=10`

## Query Parameters

- `limit`: Integer (optional, default 10) - Specifies the number of trips to be returned.

## Request Body

None.
