## GetUserTripsBySmallestDistance API

### Description
The `GetUserTripsBySmallestDistance` API retrieves user trips ordered by the smallest distance. The response is limited to the number specified by the `limit` query parameter.

### HTTP Request
#### GET
- **URL:** `http://localhost:3000/v1/trips/distance/shortest`
- **Auth:** Bearer token required

### Query Parameters

| Parameter | Type  | Description         | Default Value |
|-----------|-------|---------------------|---------------|
| `limit`   | query | Limits the number of trips returned | 10            |
