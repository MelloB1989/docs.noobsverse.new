# GetUserTripsBySmallestDistance

## Overview

This API retrieves user trips sorted by the smallest distance. The response is limited to a specified number of trips.

## HTTP Request

### GET 
`http://localhost:3000/v1/trips/distance/shortest?limit=10`

### Query Parameters

- **limit**: The maximum number of trips to return. (`default`: 10)

### Authentication

This API uses Bearer Token authentication. You must include a valid JWT token in the Authorization header of your request.
