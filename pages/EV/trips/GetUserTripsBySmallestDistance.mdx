# GetUserTripsBySmallestDistance API

This API endpoint allows you to retrieve a list of user trips sorted by the shortest distance.

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/distance/shortest`
- **Method:** GET
- **Authentication:** Bearer Token

## Query Parameters

- **limit**: (optional) Specifies the maximum number of trips to return. Defaults to 10 if not specified.

The response will contain a list of user trips sorted by the shortest distance, limited to the specified number of trips.
