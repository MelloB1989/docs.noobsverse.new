# GetUserTripsBySmallestDistance API

## Description

The `GetUserTripsBySmallestDistance` API retrieves a list of user trips sorted by the smallest distance.

## HTTP Request

### GET

- **URL**: `http://localhost:3000/v1/trips/distance/shortest?limit=10`
- **Body**: None
- **Authentication**: Bearer Token

## Query Parameters

- `limit`: The maximum number of trips to return. Default is 10.
