## getMyTrips API

The `getMyTrips` API allows you to retrieve a list of trips associated with the authenticated user.

### HTTP Method

`GET`

### Endpoint

```
http://localhost:3000/v1/trips
```

### Request

#### Headers

- `Authorization`: `Bearer <token>`

#### Body

No request body required.
