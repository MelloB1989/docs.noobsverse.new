# getMyTrips API

## Overview

The `getMyTrips` API allows you to retrieve a list of your trips. It requires Bearer authentication and provides trip details in response to an HTTP GET request.

## Endpoint

- **URL**: `http://localhost:3000/v1/trips/mytrips`
- **Method**: `GET`
- **Body**: None
