# Get My Trips API

The `getMyTrips` API allows you to retrieve the list of trips for a user. This is an HTTP GET request requiring Bearer Token authentication.

## Endpoint

`GET http://localhost:3000/v1/trips`

### URL
- **Method**: GET
- **Endpoint**: `http://localhost:3000/v1/trips`
- **Request Body**: None
- **Authentication**: Bearer Token
