## Overview

The `getMyTrips` API allows clients to retrieve a list of their trips. This is an HTTP API that requires authentication via a bearer token.

## Endpoint

- **URL:** `http://localhost:3000/v1/trips/mytrips`
- **Method:** GET
- **Authentication:** Bearer token

## Request Details

- **HTTP Method:** GET
- **URL:** `/v1/trips/mytrips`
- **Headers:**
  - `Authorization: Bearer <your_token>`

No request body is needed for this API.
