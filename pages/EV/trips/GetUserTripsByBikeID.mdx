# GetUserTripsByBikeID API

## Description

The `GetUserTripsByBikeID` API endpoint allows you to retrieve a list of trips associated with a specific bike ID.

## HTTP Request

- **Method**: `GET`
- **URL**: `http://localhost:3000/v1/trips/bike/{bikeId}`

  Replace `{bikeId}` with the actual bike ID.

## Authentication

This endpoint requires Bearer token authentication.

### Bearer Token

You must include a valid bearer token in your request headers.

- **Header**: `Authorization: Bearer {token}`

Replace `{token}` with your actual token when making requests.

## Request Body

There is no request body required for this `GET` endpoint.

## Response

On success, the server will return a list of user trips associated with the specified bike ID.
