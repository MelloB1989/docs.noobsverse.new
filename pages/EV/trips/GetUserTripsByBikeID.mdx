# GetUserTripsByBikeID

This API retrieves the trips associated with a specified bike ID. It requires bearer token authentication.

## HTTP Request

### Method

`GET`

### URL

`http://localhost:3000/v1/trips/bike/{bikeID}`

Replace `{bikeID}` in the URL with the actual bike ID, e.g., `i6bjwk0yhu`.

### Body

This endpoint does not require a request body.
