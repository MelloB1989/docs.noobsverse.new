## getAllGroups API

The `getAllGroups` API allows you to retrieve all groups from the server. This endpoint is part of our suite of APIs for managing group data.

### Endpoint

- **HTTP Method**: GET
- **URL**: `http://localhost:3000/v1/groups/getAll`
- **Authentication**: Bearer token is required

- **Headers**:

### Usage

To use this API, simply make an HTTP GET request to the URL provided with the appropriate bearer token in the headers. The server will return a list of all groups if the token is valid.

The response will contain details of all groups in JSON format. Handle the response data appropriately in your application to display or process the group information.
