# getAllGroups API

The `getAllGroups` API allows you to retrieve a list of all groups.

## Endpoint

### GET Request

- **URL:** `http://localhost:3000/v1/groups/getAll`
- **Authentication:** None required
- **Request Body:** None

### Example Request

```sh
curl -X GET "http://localhost:3000/v1/groups/getAll"
```

## Response

The response will typically include details about the groups.

## Notes

No authentication is required to access this endpoint, and it does not necessitate a request body.
