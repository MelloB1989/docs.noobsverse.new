# createGroup API

This API endpoint allows you to create a new group by sending a POST request. Below are the details required to execute this API call.

## HTTP Method

`POST`

## Endpoint

`http://localhost:3000/v1/groups/create`

## Request Body

The body of the request should be in JSON format and must include the following fields:

```json
{
  "name": "Eco Blasters",
  "description": "A group focused on being eco friendly."
}
```

### Fields

- **name**: A string representing the name of the group.
- **description**: A string providing a brief description of the group's purpose.

## Authentication

This API uses bearer token authentication. Ensure your requests include a valid bearer token in the request header to successfully authenticate. 

Make sure to replace the example token with a valid one in a real scenario.
