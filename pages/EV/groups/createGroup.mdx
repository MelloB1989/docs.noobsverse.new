## Create Group

### Description
This API endpoint allows you to create a new group.

### Endpoint
**HTTP Method:** POST  
**URL:** `http://localhost:3000/v1/groups/create`

### Request Body
**Content-Type:** application/json

```json
{
  "name": "Green Warriors",
  "description": "A group focused on reducing carbon emissions."
}
```
