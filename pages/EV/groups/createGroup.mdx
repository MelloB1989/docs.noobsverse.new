# Create Group API

This documentation provides details on how to use the `createGroup` API to create a new group.

## Endpoint Details

### HTTP Method

`POST`

### URL

`http://localhost:3000/v1/groups/create`

## Request Body

The request body should be in JSON format and include the following parameters:

```json
{
  "name": "Green Warriors",
  "description": "A group focused on reducing carbon emissions."
}
```

### Parameters

- `name` (string): The name of the group.
- `description` (string): A brief description of the group's focus.
