# createGroup

This API endpoint is used to create a new group.

## Endpoint

**POST** `http://localhost:3000/v1/groups/create`

## Headers

### Authorization

This API requires a Bearer token for authentication.

```http
Authorization: Bearer <token>
```

Example:

```plaintext
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiI1Z2VhcGdhcHhzIn0.yNyKfoOQouzSNDwp_5rJ9CAEMVJo5uqueJHNpG-a3Aw
```

## Request Body

The request body should be in JSON format.

```json
{
  "name": "Eco Blasters",
  "description": "A group focused on being eco friendly."
}
```

### Parameters

- **name**: The name of the group.
- **description**: A brief description of the group.
