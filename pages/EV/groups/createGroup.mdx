# Create Group API

This endpoint allows you to create a new group by providing necessary details like the name and description of the group.

## Endpoint Details

- **Type:** HTTP
- **Method:** POST
- **URL:** `http://localhost:3000/v1/groups/create`
- **Authentication:** Bearer Token

## Authentication

To access this endpoint, you need to include a Bearer token in the Authorization header of your request.

Include this token in your request header as follows:

```
Authorization: Bearer <token>
```

## Request Body

The request body must be in JSON format and include the following fields:

```json
{
  "name": "Eco Blasters",
  "description": "A group focused on being eco friendly."
}
```

### Fields

- **name** (string): The name of the group.
- **description** (string): A brief description of the group.
