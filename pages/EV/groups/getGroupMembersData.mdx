# getGroupMembersData API

The `getGroupMembersData` API endpoint retrieves data for members of a specific group.

## HTTP Method: GET

### URL
`http://localhost:3000/v1/groups/2/members/data`

### Request

#### Headers
- `Authorization`: Bearer token

#### Body
None

#### Authentication
