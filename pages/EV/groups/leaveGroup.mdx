# leaveGroup API

The `leaveGroup` API allows a user to leave a specific group.

## Endpoint

**GET** `http://localhost:3000/v1/groups/2/leave`

## Authentication

This endpoint requires Bearer Token authentication.

```plaintext

```

## Request Body

No request body is needed for this API.

## Response

The response will indicate whether the user has successfully left the group.
