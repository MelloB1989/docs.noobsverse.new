## leaveGroup API

The `leaveGroup` API allows a user to leave a specific group. This operation is performed using an HTTP GET request to a designated endpoint, and requires authentication via a bearer token.

### Endpoint

#### GET /v1/groups/2/leave

- **URL:** `http://localhost:3000/v1/groups/2/leave`
- **Method:** GET
- **Authentication:** Bearer Token
