# Join Group API

This document provides the details for the `joinGroup` API, which allows a user to join a specific group.

## Metadata

- **Name:** joinGroup
- **Type:** HTTP
- **Sequence:** 2

## Endpoint

This API uses a HTTP POST request.

- **URL:** `http://localhost:3000/v1/groups/vDLvtaMVs7zAVXZWkX0lL/join`
- **Method:** POST

## Request

### Body

- **Body:** None

### Authentication

This API requires Bearer token authentication.

- **Type:** Bearer Token
- **Token:**


## Usage

To use this API, make a POST request to the specified URL, including the bearer token in the request headers for authorization. No body content is needed for this request.

- **Authorization Header:**
  ```
  Authorization: Bearer <token>
  ```

Replace `<token>` with the actual bearer token provided.
