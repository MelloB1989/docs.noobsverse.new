## joinGroup API

The `joinGroup` API allows a user to join a specific group.

### HTTP Request

**POST** `http://localhost:3000/v1/groups/2/join`

No request body is required for this endpoint.

### Response

A successful response will return status `200 OK` along with the details of the group that the user has joined.
