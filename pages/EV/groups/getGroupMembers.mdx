# getGroupMembers API

The `getGroupMembers` API provides access to the members of a specific group.

## HTTP Request

### Method: GET

- **URL**: `http://localhost:3000/v1/groups/2/members`
- **Body**: None
- **Authentication**: Bearer Token

## Responses

- **200 OK**: Successfully retrieved the list of group members.
- **401 Unauthorized**: Authentication failed. Ensure the token is correctly set.
- **404 Not Found**: The specified group does not exist.
