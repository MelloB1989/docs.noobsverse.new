# getGroupMembers API

## Description

The `getGroupMembers` API endpoint retrieves the list of members in a given group.

## HTTP Method

**GET**

## Endpoint

```
http://localhost:3000/v1/groups/2/members
```

## Request

### Headers

- **Authorization**: Bearer Token

### Body

None

## Authorization

This API requires Bearer Token authentication.

### Bearer Token

Example Token:


## Response

The response will contain the list of group members.
