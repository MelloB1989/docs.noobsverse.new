# getGroupDetails API

## Overview

The `getGroupDetails` API is an HTTP GET request designed to retrieve details about a specific group. This request is made to the local server and requires authentication via a bearer token.

## HTTP Request

- **Method**: GET
- **URL**: `http://localhost:3000/v1/groups/2`
- **Body**: None
