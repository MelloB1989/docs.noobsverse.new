# getGroupDetails API

The `getGroupDetails` API allows you to retrieve details of a specific group using an HTTP GET request. This endpoint requires authentication via a Bearer token.

## Endpoint

**URL**: `http://localhost:3000/v1/groups/2`

**Method**: `GET`

**Authentication**: Bearer token

## Authentication

To access the `getGroupDetails` endpoint, a valid Bearer token must be included in the request header.

Include the token in the Authorization header as follows:

```
Authorization: Bearer <token>
```

The request does not require a body.

## Error Handling

Ensure that the Bearer token is valid and has the necessary permissions to access the group details. Invalid tokens will result in an authentication error.
