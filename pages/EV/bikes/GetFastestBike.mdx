## GetFastestBike

**Endpoint:** `GET http://localhost:3000/v1/bikes/get/speed/fastest`

**Type:** HTTP

**Authorization:** Bearer Token

## Summary

The `GetFastestBike` endpoint retrieves the fastest bike from the database. This endpoint requires a bearer token for authorization.

## Request

### URL

`http://localhost:3000/v1/bikes/get/speed/fastest`

### Method

`GET`

### Body

None

### Headers

- Authorization: Bearer token
