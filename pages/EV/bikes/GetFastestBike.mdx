## GetFastestBike API

### Overview

This API endpoint retrieves the fastest bike available in the system.

### HTTP Request

#### GET
**URL:** `http://localhost:3000/v1/bikes/get/speed/fastest`

### Request Headers

- **Authorization:** Bearer Token (required)

None.
