## GetBikesByTopSpeedGT

Fetches bikes with top speeds greater than a specified value.

### API Type

HTTP

### Sequence

8

## GET Request

### URL

`http://localhost:3000/v1/bikes/get/speed_gt/50`

### Request Body

None

### Authentication

Bearer Token

### Bearer Token Example

```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

Please ensure you replace the example token with your actual bearer token for authorization.  

### Example Request

```bash
curl -X GET "http://localhost:3000/v1/bikes/get/speed_gt/50" -H "Authorization: Bearer YOUR_BEARER_TOKEN"
```

Replace `YOUR_BEARER_TOKEN` with your actual token.

## Response

### Success

- **Code**: 200 OK
- **Content**: List of bikes with top speeds greater than the specified value

### Error

- **Code**: 401 Unauthorized
- **Content**: Unauthorized error due to missing or invalid token

Make sure your requests are authenticated using a valid bearer token.

## Summary

The `GetBikesByTopSpeedGT` API fetches bikes whose top speeds exceed a specified value. To use this API, make a GET request to the provided URL with the appropriate bearer token for authentication. Responses will vary based on whether the token is valid or not.
