# GetBikesByTopSpeedGT API

## Overview

The `GetBikesByTopSpeedGT` API fetches a list of bikes that have a top speed greater than 50 km/h. This API requires HTTP bearer authentication for access.

## HTTP Request

**GET** `http://localhost:3000/v1/bikes/get/speed_gt/50`

### Request Headers

| Key           | Value       |
| ------------- | ----------- |
| Authorization | Bearer {token} |

### Parameters

This endpoint does not require any parameters or body.

## Error Handling

The API may return error messages in case of issues such as authorization failure or server errors. Typical errors include:

- `500 Internal Server Error`: When there is an issue on the server side.
