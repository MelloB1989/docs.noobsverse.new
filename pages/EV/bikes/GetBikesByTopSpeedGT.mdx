## GetBikesByTopSpeedGT

### Description

The `GetBikesByTopSpeedGT` API endpoint retrieves a list of bikes whose top speed is greater than 50 km/h. It is a useful endpoint for applications that need to filter bikes based on their performance specifications.

### Endpoint

- **Type:** HTTP
- **Method:** GET
- **URL:** `http://localhost:3000/v1/bikes/get/speed_gt/50`
