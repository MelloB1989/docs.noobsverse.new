## Overview

The `GetBikesByTopSpeedGT` API endpoint allows users to retrieve bikes with a top speed greater than a specified value. This document provides usage details for integrating with the API.

## Endpoint

### GET /v1/bikes/get/speed_gt/50

- **URL:** `http://localhost:3000/v1/bikes/get/speed_gt/50`
- **Method:** GET
- **Authentication:** Bearer Token
- **Body:** None

## Usage

To use the `GetBikesByTopSpeedGT` endpoint, send a GET request to the URL provided above, ensuring that your request header includes a valid Bearer token for authentication.

## Response

On successful authentication, the response will include a list of bikes with a top speed greater than the specified limit. Each bike will have details such as model, brand, and speed.
