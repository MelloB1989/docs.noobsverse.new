# GetBikesByTopSpeedGT

This API endpoint retrieves a list of bikes whose top speed is greater than 50 km/h.

## HTTP Request

**GET** `/v1/bikes/get/speed_gt/50`

### Headers

- **Authorization**: Bearer {token}

## Response

A successful response will return a JSON object containing an array of bikes.

### Example Response

```json
[
  {
    "id": "bike1",
    "model": "Speedster 3000",
    "topSpeed": 55
  },
  {
    "id": "bike2",
    "model": "Windrider X",
    "topSpeed": 70
  }
]
```

## Notes

- Ensure that your bearer token is valid and has not expired.
- The `topSpeed` parameter is fixed to 50 km/h in this endpoint.
