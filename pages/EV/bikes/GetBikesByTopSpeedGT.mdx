## GetBikesByTopSpeedGT

### Description

This API endpoint retrieves a list of bikes that have a top speed greater than 50.

### HTTP Request

`GET http://localhost:3000/v1/bikes/get/speed_gt/50`

### Response Body

No request body is required for this endpoint.

### Authentication

This endpoint requires Bearer token authentication. The token must be included in the `Authorization` header of the request.

Example token:
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

### Example Request

```http
GET http://localhost:3000/v1/bikes/get/speed_gt/50
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

### Example Response

If the request is successful, the server will respond with a list of bikes. The structure of the response will depend on the implementation. Ensure to replace the token value with a valid token when making actual requests.
