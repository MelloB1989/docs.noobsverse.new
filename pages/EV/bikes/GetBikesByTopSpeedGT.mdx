# GetBikesByTopSpeedGT

Retrieve bikes with a top speed greater than 50 km/h.

## Endpoint

**GET** `http://localhost:3000/v1/bikes/get/speed_gt/50`

## Authentication

The endpoint requires Bearer token authentication.

### Bearer Token

```plaintext
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

## Request

- **URL Parameters**: None
- **Body**: None
- **Headers**:
  - `Authorization`: Bearer token as shown above

### Example Request

```http
GET http://localhost:3000/v1/bikes/get/speed_gt/50
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

## Response

### Successful Response

- **Status**: `200 OK`
- **Content Type**: `application/json`

The response will be a JSON object containing the list of bikes with a top speed greater than 50 km/h.

### Example Successful Response

```json
[
  {
    "id": "bike1",
    "name": "Speedster 3000",
    "top_speed": 55
  },
  {
    "id": "bike2",
    "name": "Flash Rider",
    "top_speed": 60
  }
]
```

### Error Responses

- **401 Unauthorized**: Missing or invalid token
- **500 Internal Server Error**: Server-side issues

## Summary

This endpoint allows clients to retrieve a list of bikes with a top speed greater than 50 km/h. Authentication using a bearer token is required. Ensure the token is included in the request headers to successfully fetch the data.
