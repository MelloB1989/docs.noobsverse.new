# GetBikesByTopSpeedGT

## Description

The `GetBikesByTopSpeedGT` API endpoint retrieves a list of bikes with a top speed greater than a specified value. This API is useful for filtering bikes based on speed criteria.

## Endpoint

- **Type**: `GET`
- **URL**: `http://localhost:3000/v1/bikes/get/speed_gt/50`

Include the following in the `Authorization` header of your request:

```
Authorization: Bearer <token>
```

Replace `<token>` with the actual token provided.

## Request

- **HTTP Method**: `GET`
- **Body**: No body required
- **Authentication Method**: Bearer Token is required in the request header.

## Response

Upon success, the API will return a JSON response containing the list of bikes with top speeds greater than the specified value.
