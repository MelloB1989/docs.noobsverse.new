# GetBikeByFrameNumber API

The `GetBikeByFrameNumber` API allows you to retrieve information about a bike using its frame number. The API uses HTTP for communication and requires bearer token authentication.

## Endpoint

### HTTP Request

`GET http://localhost:3000/v1/bikes/get/frame/{frameNumber}`

- Replace `{frameNumber}` with the actual frame number of the bike you are querying. For example, `FR123456789`.

### Authentication

This endpoint requires a bearer token for authentication. You must include the token in the `Authorization` header of your HTTP request.

### Request Body

This endpoint does not require a request body. All necessary information is provided in the URL and authorization header.

## Example Request

```http
GET /v1/bikes/get/frame/FR123456789 HTTP/1.1
Host: localhost:3000
Authorization: Bearer YOUR_BEARER_TOKEN
```

## Summary

- **API Name**: GetBikeByFrameNumber
- **Type**: HTTP
- **Sequence**: 4
- **Method**: GET
- **Authentication**: Bearer Token

This API is designed to be simple and requires only the frame number of the bike and proper authentication to function. Make sure to handle the token securely and do not expose it in unsecured environments.
