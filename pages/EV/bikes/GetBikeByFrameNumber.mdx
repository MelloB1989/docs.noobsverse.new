# GetBikeByFrameNumber

The `GetBikeByFrameNumber` API allows you to retrieve information about a bike using its frame number.

## Endpoint

**HTTP Method:** `GET`

**URL:** `http://localhost:3000/v1/bikes/get/frame/FR123456789`

## Authentication

This endpoint requires a bearer token for authentication.

**Example Bearer Token:**

```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

## Request

**Headers:**

```
Authorization: Bearer <your-bearer-token>
```

**Body:** None

## Example Request

```
GET http://localhost:3000/v1/bikes/get/frame/FR123456789
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

## Response

The response will include details about the bike corresponding to the given frame number.

_Response example:_

```json
{
  "frameNumber": "FR123456789",
  "brand": "BikeBrand",
  "model": "BikeModel",
  "color": "Red",
  "status": "available"
}
```

## Notes

- Ensure you have the correct bearer token before making a request.
- Replace `<your-bearer-token>` with the actual token in the `Authorization` header.
