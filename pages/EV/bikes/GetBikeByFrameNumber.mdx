# GetBikeByFrameNumber API

## Endpoint Description

The `GetBikeByFrameNumber` API retrieves details about a bike using the specified frame number. Authentication is required via a bearer token.

## HTTP Request

### GET

```http
GET http://localhost:3000/v1/bikes/get/frame/FR123456789
```

- **Body:** None
- **Authentication:** Bearer token
