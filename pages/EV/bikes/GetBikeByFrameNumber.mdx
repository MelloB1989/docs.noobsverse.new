# GetBikeByFrameNumber API

The `GetBikeByFrameNumber` API allows you to retrieve information about a bike using its frame number.

## API Sequence

- **Sequence**: 4

## HTTP Request

### GET

- **URL**: `http://localhost:3000/v1/bikes/get/frame/FR123456789`
- **Method**: `GET`
- **Request Body**: None

### Authentication

- **Type**: Bearer Token
- **Token**: `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0`

## Description

This endpoint retrieves data related to a bike identified by its unique frame number `FR123456789`. The request must include a valid bearer token to authorize access to the bike's data.

Make sure to replace the frame number and token with the actual data you need to query and authenticate your requests.
