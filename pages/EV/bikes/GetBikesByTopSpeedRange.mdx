## GetBikesByTopSpeedRange

The `GetBikesByTopSpeedRange` API allows you to fetch a list of bikes within a specified top speed range.

- **Type:** HTTP

- **URL:** `http://localhost:3000/v1/bikes/get/speed/50/47`
