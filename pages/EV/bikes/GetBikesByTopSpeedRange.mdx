## GetBikesByTopSpeedRange

The `GetBikesByTopSpeedRange` API retrieves a list of bikes within a specific top speed range.

### HTTP Request

**GET** `http://localhost:3000/v1/bikes/get/speed/50/47`

### Authentication

This API requires a Bearer Token for authentication.

**Auth:** Bearer `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0`

### Request Body

This API does not require a request body.

### Response

The API returns a JSON object containing the list of bikes that fall within the specified speed range.

### Errors

Possible error responses include:

- **401 Unauthorized**: The token provided is invalid or expired.
- **404 Not Found**: No bikes found within the specified speed range.
