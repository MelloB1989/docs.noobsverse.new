# GetBikesByTopSpeedRange API

## Description
The `GetBikesByTopSpeedRange` API allows you to fetch details of bikes that fall within a specified speed range.

## Endpoint
**URL:** `http://localhost:3000/v1/bikes/get/speed/{upperSpeed}/{lowerSpeed}`

| Method | URL                                                  |
|--------|------------------------------------------------------|
| GET    | `http://localhost:3000/v1/bikes/get/speed/50/47`     |

Ensure you replace `{upperSpeed}` and `{lowerSpeed}` with appropriate values.

## Request
### Headers
| Key       | Value              |
|-----------|--------------------|
| Authorization | Bearer `{{token}}` |

### Auth
**Type:** Bearer Token

### Body
This endpoint does not require a request body.

## Response
The response will be a JSON object containing the bikes' details that fall within the specified speed range.

Example:

```json
{
  "data": [
    {
      "id": "bike1",
      "make": "BrandA",
      "model": "ModelX",
      "topSpeed": 49
    },
    {
      "id": "bike2",
      "make": "BrandB",
      "model": "ModelY",
      "topSpeed": 48
    }
  ]
}
```

## Notes
- Replace `{{token}}` with your actual bearer token in the Authorization header.
- Ensure the speed values in the URL path are integers.
