# GetBikesByTopSpeedRange API

## Overview

The `GetBikesByTopSpeedRange` API allows you to retrieve a list of bikes within a specific top speed range. It is an HTTP GET endpoint that requires a bearer token for authentication.

## HTTP Request

- **Method:** GET
- **URL:** `http://localhost:3000/v1/bikes/get/speed/50/47`
- **Body:** None

## Authentication

This endpoint requires a bearer token for authentication. The token must be included in the header of the HTTP request.


```bash
curl -X GET http://localhost:3000/v1/bikes/get/speed/50/47 \
     -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0"
```

## Response

The response will be a JSON object containing the list of bikes that fall within the specified top speed range.

Example:

```json
{
  "bikes": [
    {
      "id": "bike1",
      "model": "Speedster",
      "topSpeed": 48
    },
    {
      "id": "bike2",
      "model": "Flash",
      "topSpeed": 49
    }
  ]
}
```

## Errors

- **401 Unauthorized:** The bearer token is missing or invalid.
- **404 Not Found:** No bikes were found within the specified speed range.
- **500 Internal Server Error:** An unexpected error occurred on the server.
