# GetBikesByTopSpeedRange API

## Description

Fetches bikes within a specific top speed range.

## HTTP Request

### Method: GET
### URL
`http://localhost:3000/v1/bikes/get/speed/50/47`

### Request Body
No request body required.

### Authentication

This endpoint uses bearer token authentication.

**Authorization Header Format:**
```
Authorization: Bearer <token>
```

**Example Token:**
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

### Response

The response will contain the list of bikes within the specified top speed range.
