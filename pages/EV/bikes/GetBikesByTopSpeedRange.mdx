# GetBikesByTopSpeedRange

The `GetBikesByTopSpeedRange` API allows you to retrieve a list of bikes within a specified top speed range. This API is accessed via a `GET` request and requires bearer token authentication.

## Endpoint

- **HTTP Method**: GET
- **URL**: `http://localhost:3000/v1/bikes/get/speed/50/47`
- **Body**: None

## Authentication

This API uses bearer token authentication. You need to include a valid token in the `Authorization` header of your request.
