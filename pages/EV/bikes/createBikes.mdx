## Create Bikes API

The `createBikes` endpoint allows you to add new bikes to the system.

### HTTP Method

`POST`

### URL

`http://localhost:3000/v1/bikes/create`

### Authentication

No authentication required. 

### Headers

```plaintext
X-Karma-Admin-Auth: CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab
```

### Request Body

The request body should be in JSON format and contain a list of bike objects with the following fields:

- `frame_number` (String): The unique frame number of the bike.
- `name` (String): The name of the bike.
- `station_id` (String): The ID of the station where the bike is located.
- `top_speed` (Number): The top speed of the bike in km/h.
- `range` (Number): The range of the bike in km.
- `time_to_station` (Number): The estimated time to the station in minutes.
- `images` (Array of Strings): URLs of images of the bike.
