# Create Bikes API

## Endpoint Details

**URL:** `http://localhost:3000/v1/bikes/create`  
**Method:** `POST`  
**Authentication:** None

## Headers

Include the following headers in the request:

```http
X-Karma-Admin-Auth: CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab  
```

## Request Body (JSON)

The body of the request should be a JSON array containing one or more bike objects. Each bike object must have the following fields:

- `frame_number` (string): Unique identifier for the bike's frame.
- `name` (string): Name of the bike.
- `station_id` (string): ID of the station where the bike is located.
- `top_speed` (number): Maximum speed of the bike (km/h).
- `range` (number): Maximum range of the bike (km).
- `time_to_station` (number): Estimated time to reach the station (minutes).
- `images` (array of strings): URLs to images of the bike.
