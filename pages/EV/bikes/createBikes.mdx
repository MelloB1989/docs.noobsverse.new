# Create Bikes API

The **Create Bikes API** allows you to add new bikes to the database. Here's how to use the endpoint effectively.

## Endpoint

- **URL**: `http://localhost:3000/v1/bikes/create`
- **Method**: `POST`
- **Authentication**: None required

## Headers

- `X-Karma-Admin-Auth`: `CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab`

## Request Body

The request body should be in JSON format, containing an array of bike objects. Each bike object must have the following fields:

- `frame_number` (string): Unique identifier for the bike.
- `name` (string): Name of the bike.
- `station_id` (string): Identifier for the station where the bike is located.
- `top_speed` (number): Maximum speed the bike can achieve (in km/h).
- `range` (number): The maximum distance the bike can cover on a full charge (in km).
- `time_to_station` (number): Estimated time in minutes to reach the nearest station.
- `images` (array of strings): URLs of the images representing the bike.

- The `images` array should contain valid URLs pointing to images representing the bike.
