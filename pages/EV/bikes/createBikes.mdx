# createBikes API

## Overview
The `createBikes` API allows you to create multiple bike records in the system.

## Endpoint Details
- **Type:** HTTP
- **Sequence:** 1

## POST Request
- **URL:** `http://localhost:3000/v1/bikes/create`
- **Body Format:** JSON
- **Authentication:** None

## Headers
Include the following headers in your request:
```plaintext
X-Karma-Admin-Auth: CoffeeCodesMelloBxAparnaEewqjbdkwqbkdjoehui3r7328grehjvdjvchjiuiudhsjkhkjsadab
```
