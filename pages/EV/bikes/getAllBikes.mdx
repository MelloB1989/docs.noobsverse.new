## getAllBikes

### Overview

The `getAllBikes` API endpoint is used to retrieve all available bikes from the server.

### HTTP Request

- **URL:** `http://localhost:3000/v1/bikes/get`
- **Method:** `GET`
- **Authentication:** Bearer Token
