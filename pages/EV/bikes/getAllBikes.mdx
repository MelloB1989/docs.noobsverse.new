## getAllBikes

The `getAllBikes` API is an HTTP GET endpoint used to retrieve information on all bikes available.

### Endpoint

- **URL:** `http://localhost:3000/v1/bikes/get`
- **Method:** `GET`

### Request

- **Body:** None
- **Authorization:** Bearer Token
