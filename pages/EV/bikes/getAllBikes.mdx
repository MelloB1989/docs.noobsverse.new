# getAllBikes API

The `getAllBikes` API allows you to retrieve all available bike data from the server. It is a simple HTTP GET request that requires bearer token authentication.

## API Endpoint

- **URL:** `http://localhost:3000/v1/bikes/get`
- **Method:** `GET`
- **Authentication:** Bearer Token

The `getAllBikes` API will respond with a list of all bikes available on the server. The format and structure of the response may vary depending on the server implementation. Typically, it should be in JSON format, providing the necessary details for each bike.
