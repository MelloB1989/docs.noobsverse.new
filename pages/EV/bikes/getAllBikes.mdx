# getAllBikes API

The `getAllBikes` API is used to retrieve a list of all bikes.

## Endpoint

`GET http://localhost:3000/v1/bikes/get`

## Request

### URL
`http://localhost:3000/v1/bikes/get`

### Method
`GET`

### Body
None

### Authentication
Bearer token is required.

```
Authorization: Bearer <token>
```
