## GetBikesByTopSpeedLT API

This API endpoint retrieves a list of bikes that have a top speed less than a specified value.

### HTTP Request

**GET** `http://localhost:3000/v1/bikes/get/speed_lt/50`

- **URL**: `http://localhost:3000/v1/bikes/get/speed_lt/50`
- **Method**: GET
- **Request Body**: None
- **Authorization**: Bearer Token

### Authorization

To use this endpoint, you must provide a bearer token.
