## Overview

The `GetBikesByTopSpeedLT` API is an HTTP-based service that retrieves a list of bikes with a top speed less than a specified limit. This document provides detailed information on how to interact with this API.

## API Endpoint

- **URL:** `http://localhost:3000/v1/bikes/get/speed_lt/50`
- **Method:** GET
- **Authentication:** Bearer Token is required.

## Request Details

- **HTTP Method:** GET
- **Request Body:** None

This endpoint does not require any parameters in the request body. It relies entirely on the URL and authentication header to retrieve the appropriate data.

## Usage

Make a GET request to the specified URL with the appropriate Bearer Token in the authorization header to receive a list of bikes with top speeds below the specified threshold (in this case, 50).
