# GetBikesByTopSpeedLT API

## Description

Retrieve bikes with a top speed less than 50 km/h.

## HTTP Request

### Method

`GET`

### URL

`http://localhost:3000/v1/bikes/get/speed_lt/50`

## Request Body

None
