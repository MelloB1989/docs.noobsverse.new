## Overview

The `GetBikesByTopSpeedLT` API endpoint retrieves a list of bikes with a top speed less than a specified value. This API supports HTTP `GET` requests and requires authentication using a bearer token.

## Endpoint Information

- **URL:** `http://localhost:3000/v1/bikes/get/speed_lt/50`
- **Method:** `GET`
- **Authentication:** Bearer token

## Request Details

### URL

```plaintext
http://localhost:3000/v1/bikes/get/speed_lt/50
```

### Method

```plaintext
GET
```

### Request Body

- **Body:** None. This endpoint does not require a request body.

### Authentication

This endpoint requires bearer token authentication. The client must include a valid authorization header with a bearer token in each request.

### Authentication Header Format

```plaintext
Authorization: Bearer <token>
```

```plaintext
Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

## Response Details

The response from the server will contain the list of bikes with a top speed less than the specified value of 50.

Host: localhost:3000
