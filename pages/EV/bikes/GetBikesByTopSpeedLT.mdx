# GetBikesByTopSpeedLT API

## Endpoint Description

The GetBikesByTopSpeedLT API retrieves a list of bikes whose top speed is less than a specified value.

## HTTP Request

### Method

`GET`

### URL

`http://localhost:3000/v1/bikes/get/speed_lt/50`

### Body

No request body is required.
