# GetBikesByTopSpeedLT API

This API endpoint retrieves a list of bikes with a top speed less than 50 km/h.

## Endpoint

### GET Request

- **URL:** `http://localhost:3000/v1/bikes/get/speed_lt/50`
- **Request Body:** None
- **Authentication:** Bearer Token
