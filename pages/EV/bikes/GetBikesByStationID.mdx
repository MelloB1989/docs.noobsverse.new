## GetBikesByStationID API

The `GetBikesByStationID` API retrieves a list of bikes available at a specific station.

### Meta

- **Name**: GetBikesByStationID
- **Type**: HTTP
- **Sequence**: 5

### HTTP Request

- **Method**: GET
- **URL**: `https://ev-backend.app.k8s.coffeecodes.in/v1/bikes/get/station/e6j_2fyc23`
- **Request Body**: None
- **Authorization**: Bearer Token
