# GetBikesByStationID API

This document provides details about the `GetBikesByStationID` API endpoint, including how to authenticate and make a request.

## Overview

- **API Name**: GetBikesByStationID
- **Type**: HTTP
- **Sequence**: 5

## Endpoint

- **HTTP Method**: GET
- **URL**: `https://ev-backend.app.k8s.coffeecodes.in/v1/bikes/get/station/{station_id}`
