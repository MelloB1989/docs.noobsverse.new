# GetBikesByStationID API

This API retrieves a list of bikes available at a specific station.

## HTTP Request

### GET

```http
GET https://ev-backend.app.k8s.coffeecodes.in/v1/bikes/get/station/e6j_2fyc23
```

### Request Body

- None

### Authentication

This API requires a bearer token for authentication.

```plaintext
Authorization: Bearer <token>
```

## Example Bearer Token

```plaintext
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0
```

## Usage

```bash
curl -X GET "https://ev-backend.app.k8s.coffeecodes.in/v1/bikes/get/station/e6j_2fyc23" \
-H "Authorization: Bearer <token>"
```

Replace `<token>` with your actual bearer token.
