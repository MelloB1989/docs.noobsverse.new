# GetBikesByStationID API

The `GetBikesByStationID` API allows you to retrieve a list of bikes available at a specific station.

- Sequence: 5

- **URL**: `https://ev-backend.app.k8s.coffeecodes.in/v1/bikes/get/station/e6j_2fyc23`
- **Body**: None
