## getBikeByID API

This API retrieves information about a bike using its ID. It requires a Bearer token for authentication.

### Endpoint

`GET http://localhost:3000/v1/bikes/get/jruefu191e`

### Authentication

This endpoint requires Bearer Token authentication.


### Request

- **Method**: `GET`
- **URL**: `http://localhost:3000/v1/bikes/get/jruefu191e`
- **Body**: Not required
- **Headers**: 
  - `Authorization`: `Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJwaG9uZSI6Iis5MTk4MTI5NDA3MDYiLCJ1aWQiOiJhZnJ3dXZkXzFqIn0.se2WRRJhST9uvOyhTwunvJiiMYD-RYfKqfXPMN2NnE0`

The response will contain the details of the bike. Below is a sample response structure:

```json

  "model": "Mountain Bike",
  "brand": "XYZ",
  "status": "Available"

```

To use this API, make a GET request to the specified URL while including the Bearer token in the Authorization header.
