# getAllBikes API Documentation

The `getAllBikes` API endpoint allows you to retrieve a list of all bikes.

## HTTP Request

`GET http://localhost:3000/v1/bikes/get`

## Response Format

The response is a JSON array containing objects with the following fields:

- **id**: Unique identifier for the bike.
- **brand**: Brand of the bike.
- **model**: Model of the bike.
- **status**: Availability status of the bike (e.g., `available`, `unavailable`).
